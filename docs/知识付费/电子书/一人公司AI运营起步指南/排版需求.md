## 三、封面设计描述

### 封面信息

| 项目 | 内容 |
|------|------|
| 书名 | 一人公司 AI 运营起步指南 |
| 副标题 | 一个人，一套AI工具，一份可持续的事业 |
| 作者 | [作者名称] |
| 定价标签 | 9.9元 |

### 封面设计要求（供 AI 生成参考）

**设计风格：**
- 现代、简洁、专业
- 科技感与人文温度结合
- 适合职场/商业类书籍

**配色方案：**
- 主色调：深蓝色（#1E3A5F）或科技蓝（#2563EB）
- 辅助色：白色、浅灰色
- 点缀色：橙色或金色（突出重点）

**视觉元素建议：**
1. **方案A - 极简风格**：
   - 纯色背景 + 大字书名
   - 右下角添加简约的 AI 相关图标（如芯片、神经网络、代码符号）
   - 底部显示作者名称

2. **方案B - 场景风格**：
   - 一个人的剪影坐在电脑前工作
   - 背景是城市天际线或抽象的数据流动图案
   - 强调"一人"的概念

3. **方案C - 符号风格**：
   - 中心是一个"1"的符号，周围环绕 AI 元素
   - 表达"一人 + AI = 公司"的概念

**AI 生成封面提示词（英文）：**

```
Book cover design for "AI Operations Guide for One-Person Company", 
modern minimalist style, professional business book, 
deep blue background with tech elements, 
clean typography, suitable for digital download product, 
high quality, 8K resolution, aspect ratio 2:3
```

**AI 生成封面提示词（中文）：**

```
电子书封面设计，书名《一人公司 AI 运营起步指南》，
现代简约风格，商务专业感，深蓝色科技背景，
整洁的排版，适合数字产品，高质量，8K分辨率，比例2:3
```

**封面尺寸规格：**
- 推荐尺寸：1410 x 2250 像素（2:3 比例）
- 最小尺寸：600 x 900 像素
- 格式：PNG（透明背景）或 JPG

## 四、排版需求清单

### 4.1 整体版式

| 项目 | 规格 |
|------|------|
| 页面尺寸 | A4（210mm x 297mm）或 16开 |
| 页边距 | 上下 25mm，左右 20mm |
| 文字方向 | 从左到右，横向排版 |
| 分栏 | 单栏 |

### 4.2 字体规范

| 元素 | 字体 | 字号 | 颜色 |
|------|------|------|------|
| 封面书名 | 思源黑体 Bold / 方正兰亭黑 | 48pt | 深蓝 #1E3A5F |
| 封面副标题 | 思源黑体 Regular | 18pt | 灰色 #666666 |
| 章节标题 | 思源黑体 Bold | 24pt | 深蓝 #1E3A5F |
| 一级标题（H1） | 思源黑体 Medium | 20pt | 深蓝 #1E3A5F |
| 二级标题（H2） | 思源黑体 Medium | 16pt | 深蓝 #1E3A5F |
| 三级标题（H3） | 思源黑体 Regular | 14pt | 深蓝 #1E3A5F |
| 正文 | 思源宋体 Regular / 苹方-简 | 12pt | 黑色 #333333 |
| 引用文字 | 思源宋体 Italic | 11pt | 灰色 #666666 |
| 代码块 | JetBrains Mono / Source Code Pro | 10pt | 深灰 #444444 |
| 页码 | 思源黑体 Light | 10pt | 灰色 #999999 |

### 4.3 段落格式

| 项目 | 规格 |
|------|------|
| 行间距 | 1.5 倍行距（正文） |
| 段间距 | 段前 6pt，段后 6pt |
| 首行缩进 | 2 字符（正文段落） |
| 对齐方式 | 两端对齐 |

### 4.4 页眉页脚

| 项目 | 内容 |
|------|------|
| 页眉左侧 | 书名：《一人公司 AI 运营起步指南》 |
| 页眉右侧 | 当前章节标题 |
| 页眉线 | 0.5pt 细线，灰色 |
| 页脚左侧 | 作者名称 |
| 页脚右侧 | 页码 |
| 页脚线 | 无 |

### 4.5 特殊元素

**引用块：**
- 左侧 3pt 蓝色竖线
- 背景色：浅灰 #F5F5F5
- 左右内边距：12pt
- 上下内边距：8pt

**代码块：**
- 背景色：深灰 #2D2D2D
- 文字颜色：浅色 #E0E0E0
- 圆角：4pt
- 内边距：12pt

**提示框：**
- 左侧 4pt 蓝色竖线
- 背景色：浅蓝 #EBF5FF
- 图标：💡 或 📌

**列表样式：**
- 无序列表：圆点（•）
- 有序列表：阿拉伯数字（1. 2. 3.）
- 缩进：2 字符
