## v3.1 - 5个垂直领域选题规划

### 目标

完成5个垂直领域的选题库建设，每个领域≥50个选题

### 垂直领域选择

根据战略目标中收入拆解，选定以下5个垂直领域：

| 领域 | 定位 | 目标平台 | 预计占比 |
|------|------|----------|----------|
| AI工具评测 | 分享最新AI工具使用技巧 | 知乎、小红书 | 25% |
| 效率提升方法论 | 个人成长与生产力提升 | 公众号、知乎 | 25% |
| 副业变现指南 | 副业赚钱思路与实操 | 小红书、知乎 | 20% |
| 科技前沿解读 | 科技趋势分析与评论 | 知乎、B站 | 15% |
| 生活方式 | 品质生活、个人成长 | 小红书 | 15% |

v3.1.1 (确认选题方向)
    │

### 依赖关系

- v3.1.1 是所有选题任务的前置条件

- v3.2.1 (知识付费) 是核心，可与 v3.2.2、v3.2.3 并行执行
