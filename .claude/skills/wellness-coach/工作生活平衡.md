# 工作生活平衡

## 功能概述

帮助用户设定工作与生活的边界，跟踪工作时长和休息情况，推荐合理的工作节奏，促进健康的平衡。

## 平衡指数评估

| 指数 | 状态 | 说明 | 建议行动 |
|------|------|------|----------|
| 1-3 | 严重失衡 | 工作严重挤压生活，健康风险高 | 立即调整工作计划，增加休息 |
| 4-5 | 轻度失衡 | 工作占主导，生活质量下降 | 减少工作时间，安排个人时间 |
| 6-7 | 基本平衡 | 工作、生活大致平衡 | 维持当前节奏，留意变化 |
| 8-9 | 良好平衡 | 工作生活兼顾 | 保持良好状态 |
| 10 | 理想平衡 | 工作生活完美协调 | 作为理想状态维持 |

## 边界设定

### 时间边界

| 边界类型 | 建议设定 | 实施方法 |
|----------|----------|----------|
| 工作开始时间 | 固定每天开始时间 | 创建日历事件，设置提醒 |
| 工作结束时间 | 固定每天结束时间 | 到点停止工作，关闭设备 |
| 午餐休息 | 至少30-60分钟 | 安排到日历，离开工作区 |
| 周末休息 | 每周至少1天完全休息 | 安排非工作活动 |
| 假期规划 | 每季度至少2-3天 | 提前安排到日历 |

### 空间边界

| 边界类型 | 建议做法 |
|----------|----------|
| 物理空间 | 将工作区与生活区分离（如可能） |
| 数字边界 | 非工作时间关闭工作通知 |
| 设备使用 | 睡前1小时不使用工作设备 |
| 信息流 | 设置非工作时间的自动回复 |

## 工作时长追踪

### 每日记录内容

```
- 工作开始时间
- 工作结束时间
- 工作总时长（分钟）
- 休息次数和时长
- 工作时段（如：上午、下午）
```

### 每周分析内容

```
- 周工作总时长
- 平均每日工作时长
- 超时天数（超过8小时）
- 休息日数量
- 工作与个人时间比例
```

## 休息提醒系统

### 自动提醒触发条件

```
连续工作超过2小时：提醒短暂休息（5-10分钟）
连续工作超过4小时：强制建议长时间休息（30分钟）
工作超过8小时：提醒停止工作
周末工作：提醒今天是休息日
```

### 休息建议内容

| 休息时长 | 推荐活动 |
|----------|----------|
| 5-10分钟 | 站立拉伸、眺望远方、喝杯水 |
| 15-30分钟 | 快步走、简单运动、听音乐 |
| 30分钟以上 | 非工作相关活动、正念练习、社交 |

## 兴趣爱好追踪

### 兴趣类型

| 类型 | 示例 | 益处 |
|------|------|------|
| 运动 | 跑步、瑜伽、健身 | 释放压力，强健体魄 |
| 艺术 | 绘画、音乐、写作 | 情感表达，心理疗愈 |
| 社交 | 朋友聚会、社群活动 | 缓解孤独感，获得支持 |
| 学习 | 新技能、阅读、课程 | 持续成长，满足好奇心 |
| 自然 | 徒步、园艺、观鸟 | 连接自然，平静心灵 |

### 记录格式

```
在 WellnessRecord 中记录：
- 本周参与的兴趣活动
- 每种活动的频率
- 活动带来的愉悦感评分 (1-10)
```

## 一人公司特有挑战

### 边界模糊

**问题**：没有外部"打卡下班"的信号

**解决方案**：
- 设定明确的下班时间提醒
- 创建"下班仪式"（如关闭工作应用、收拾桌面）
- 让家人或朋友提醒自己下班

### 角色过载

**问题**：一人承担所有角色（CEO、员工、财务等）

**解决方案**：
- 明确每日角色分配（如：上午专注开发，下午处理事务）
- 学会外包（如：外包会计、客服）
- 设定每周"不工作日"

### 缺乏切换信号

**问题**：没有外部提示帮助切换状态

**解决方案**：
- 使用环境变化（如：换衣服、换房间）
- 设定过渡仪式（如：工作前喝咖啡，下班前总结）
- 使用时间块明确标注

## 使用示例

### 示例 1：检查今日工作时长

```
用户: "我今天工作了多久？"

wellness-coach 处理:
1. 查询今日的工作开始和结束时间记录
2. 计算工作总时长
3. 对比健康标准（如每日建议不超过8小时）
4. 提供平衡建议
```

### 示例 2：休息提醒

```
用户: "我连续工作4小时了"

wellness-coach 处理:
1. 识别连续工作时长达4小时（重度）
2. 立即建议休息30分钟
3. 推荐休息活动（快走、拉伸、正念）
4. 建议休息后重新评估工作计划
```

### 示例 3：周末工作提醒

```
用户: "周末我想工作"

wellness-coach 处理:
1. 查询本周已工作时长
2. 如果已超过40小时，建议休息
3. 如果需要工作，建议限制时长（如最多2小时）
4. 推荐周末的其他活动来平衡
```

### 示例 4：分析工作生活平衡

```
用户: "我最近工作生活平衡怎么样？"

wellness-coach 处理:
1. 查询最近一周的 WellnessRecord
2. 分析工作时长与休息日的比例
3. 检查社交活动和兴趣爱好的频率
4. 计算平衡指数
5. 提供改进建议
```

## 最佳实践

1. **明确边界** - 清晰地设定并坚持工作时间和休息时间
2. **主动休息** - 不要等到精疲力竭才休息
3. **多样化生活** - 保持兴趣爱好和社交活动
4. **定期评估** - 每周检查工作生活平衡状况
5. **灵活调整** - 根据实际情况调整边界

## 与其他模块的协作

### task-manager
- 工作量过重时建议任务优先级调整
- 非工作时间暂停新任务创建提醒

### schedule-manager
- 协调安排休息时间段到日程
- 设定工作开始/结束时间的自动提醒

### 压力管理.md
- 工作生活失衡是重要压力源
- 识别因过度工作导致的压力

### 情绪追踪.md
- 工作生活不平衡通常反映在情绪上
- 连续工作日的情绪追踪预警

## 健康标准参考

| 标准 | 建议值 | 说明 |
|--------|----------|------|
| 每日工作时长 | ≤ 8小时 | 超过增加倦怠风险 |
| 每周工作时长 | ≤ 40小时 | 创业者常有例外，但需注意 |
| 休息间隔 | 每2小时休息5-10分钟 | 保持精力的关键 |
| 周末休息 | 至少1天完全休息 | 恢复和充电必要 |
| 年假天数 | 至少2周/年 | 长期恢复需要 |

## 警示信号

当出现以下信号时，工作生活平衡已严重受损：

1. **持续失眠** - 工作占据睡眠时间
2. **情绪恶化** - 持续的负面情绪
3. **社交退缩** - 完全放弃社交活动
4. **健康问题** - 出现身体症状
5. **兴趣丧失** - 对以往爱好失去兴趣

出现以上任何信号，必须立即采取行动调整平衡。
