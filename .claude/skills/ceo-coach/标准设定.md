# 标准设定

> **重要**：标准应存储在 Memory 中动态读取，本文档仅作为模板参考。

## 动态标准获取

```
执行前必须：
1. 从 Memory 读取 standard-* 实体
2. 获取最新标准值
3. 参考本文档模板理解结构
4. 执行评估
```

## Memory 实体模板

### 创建标准实体

```
mcp__memory__create_entities:
{
  "entities": [{
    "name": "standard-coach-v1",
    "entityType": "Standard",
    "observations": [
      "version: 1.0.0",
      "decisionQuality_base: 80%",
      "decisionQuality_good: 90%",
      "decisionQuality_excellent: 95%",
      "coordinationEfficiency_base: 按时",
      "coordinationEfficiency_good: 提前10%",
      "coordinationEfficiency_excellent: 提前20%",
      "updatedAt: YYYY-MM-DD"
    ]
  }]
}
```

### 读取当前标准

```
mcp__memory__open_nodes:
names: ["standard-coach-v1"]
```
