# 核心指标定义文档

> 内容运营核心指标定义与采集规范
> 版本：V1.0 | 创建日期：2026-02-27

---

## 一、指标体系概述

### 1.1 目的

本文档定义内容运营的核心指标体系，为数据看板提供统一的指标口径和计算方法，确保数据采集和分析的一致性。

### 1.2 指标维度划分

| 维度 | 英文 | 说明 |
|------|------|------|
| 流量维度 | Traffic | 内容曝光和阅读相关指标 |
| 互动维度 | Engagement | 用户互动行为相关指标 |
| 转化维度 | Conversion | 用户行为转化相关指标 |
| 收入维度 | Revenue | 营收相关指标 |

---

## 二、流量维度指标

### 2.1 阅读量

| 属性 | 值 |
|------|-----|
| **指标名称** | 阅读量 |
| **英文名称** | Views |
| **定义** | 用户打开并浏览内容的次数 |
| **计算方式** | 平台统计的阅读/浏览次数总和 |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 次 |

**平台差异说明**：

| 平台 | 指标名称 | 特殊说明 |
|------|----------|----------|
| 知乎 | 阅读量 | 用户点入即计 |
| 简书 | 阅读量 | 用户点入即计 |
| CSDN | 阅读量 | 用户点入即计 |
| 掘金 | 阅读量 | 用户点入即计 |
| 微信公众号 | 阅读量 | 同一用户多次阅读计多次 |
| 头条号 | 阅读量 | 推荐流量阅读数 |
| B站 | 播放量 | 用户播放即计 |

### 2.2 曝光量

| 属性 | 值 |
|------|-----|
| **指标名称** | 曝光量 |
| **英文名称** | Impressions |
| **定义** | 内容被展示给用户的次数 |
| **计算方式** | 平台统计的内容展示次数 |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 次 |

**适用平台**：头条号（推荐曝光）、公众号（推送曝光）、B站（推荐曝光）

### 2.3 点击率

| 属性 | 值 |
|------|-----|
| **指标名称** | 点击率 |
| **英文名称** | CTR (Click-Through Rate) |
| **定义** | 用户点击内容的比例 |
| **计算方式** | `点击率 = 阅读量 / 曝光量 × 100%` |
| **采集频率** | 每日 |
| **数据来源** | 计算得出 |
| **单位** | % |

## 三、互动维度指标

### 3.1 点赞数

| 属性 | 值 |
|------|-----|
| **指标名称** | 点赞数 |
| **英文名称** | Likes |
| **定义** | 用户对内容点赞的数量 |
| **计算方式** | 平台统计的点赞次数总和 |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 次 |

### 3.2 评论数

| 属性 | 值 |
|------|-----|
| **指标名称** | 评论数 |
| **英文名称** | Comments |
| **定义** | 用户对内容评论的数量 |
| **计算方式** | 平台统计的评论条数总和（不含回复） |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 条 |

### 3.3 收藏数

| 属性 | 值 |
|------|-----|
| **指标名称** | 收藏数 |
| **英文名称** | Favorites |
| **定义** | 用户收藏内容的数量 |
| **计算方式** | 平台统计的收藏次数总和 |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 次 |

### 3.4 分享数

| 属性 | 值 |
|------|-----|
| **指标名称** | 分享数 |
| **英文名称** | Shares |
| **定义** | 用户分享内容的数量 |
| **计算方式** | 平台统计的分享次数总和 |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 次 |

### 3.5 互动率

| 属性 | 值 |
|------|-----|
| **指标名称** | 互动率 |
| **英文名称** | Engagement Rate |
| **定义** | 用户与内容互动的比例 |
| **计算方式** | `互动率 = (点赞数 + 评论数 + 收藏数 + 分享数) / 阅读量 × 100%` |
| **采集频率** | 每日 |
| **数据来源** | 计算得出 |
| **单位** | % |

**互动率分级标准**：

| 级别 | 互动率范围 | 评价 |
|------|------------|------|
| 优秀 | ≥ 5% | 表现优异 |
| 良好 | 3% ~ 5% | 表现良好 |
| 一般 | 1% ~ 3% | 需关注 |
| 较差 | < 1% | 需预警 |

## 四、转化维度指标

### 4.1 粉丝增量

| 属性 | 值 |
|------|-----|
| **指标名称** | 粉丝增量 |
| **英文名称** | Follower Growth |
| **定义** | 每日新增粉丝数减去取关数 |
| **计算方式** | `粉丝增量 = 新增粉丝数 - 取关数` |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台 |
| **单位** | 人 |

### 4.2 转化数

| 属性 | 值 |
|------|-----|
| **指标名称** | 转化数 |
| **英文名称** | Conversions |
| **定义** | 完成目标行为的用户数 |
| **计算方式** | 平台统计的转化行为次数 |
| **采集频率** | 每日 |
| **数据来源** | 各内容平台后台/自定义跟踪 |
| **单位** | 次 |

**转化目标定义**：

| 转化类型 | 定义 | 适用场景 |
|----------|------|----------|
| 引流转化 | 用户点击外链/二维码 | 私域引流 |
| 注册转化 | 用户完成注册 | 知识付费 |
| 购买转化 | 用户完成付费 | 课程/产品销售 |

### 4.3 转化率

| 属性 | 值 |
|------|-----|
| **指标名称** | 转化率 |
| **英文名称** | Conversion Rate |
| **定义** | 完成转化的用户占阅读量的比例 |
| **计算方式** | `转化率 = 转化数 / 阅读量 × 100%` |
| **采集频率** | 每日 |
| **数据来源** | 计算得出 |
| **单位** | % |

**转化率分级标准**：

| 级别 | 转化率范围 | 评价 |
|------|------------|------|
| 优秀 | ≥ 5% | 表现优异 |
| 良好 | 3% ~ 5% | 表现良好 |
| 一般 | 1% ~ 3% | 需关注 |
| 较差 | < 1% | 需预警 |

## 六、汇总指标

### 6.1 综合评分

| 属性 | 值 |
|------|-----|
| **指标名称** | 综合评分 |
| **英文名称** | Overall Score |
| **定义** | 多维度指标加权综合评估 |
| **计算方式** | 详见「看板设计文档」 |
| **采集频率** | 每日 |
| **数据来源** | 计算得出 |
| **单位** | 分 |

**评分权重**：

| 维度 | 权重 | 说明 |
|------|------|------|
| 流量 | 30% | 阅读量/曝光量 |
| 互动 | 30% | 互动率 |
| 转化 | 25% | 转化率 |
| 收入 | 15% | 营收 |

## 七、指标采集频率

| 指标类别 | 采集频率 | 采集时间 | 说明 |
|----------|----------|----------|------|
| 日常指标 | 每日 | 21:00 | 阅读量、点赞、评论等 |
| 转化指标 | 每日 | 21:00 | 转化数、转化率 |
| 收入指标 | 每日 | 21:00 | 营收、订单数 |
