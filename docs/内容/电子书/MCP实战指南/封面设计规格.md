## 设计建议

### 配色方案

**主色调**：科技蓝 `#2563EB`

- 代表 AI、科技、专业性
- 与 Claude 品牌（Anthropic）呼应

**辅助色**：

| 颜色 | 色值 | 用途 |
|------|------|------|
| 深蓝 | `#1E40AF` | 标题背景、层次 |
| 浅蓝 | `#DBEAFE` | 装饰元素、光效 |
| 橙色 | `#F97316` | 强调、CTA 元素 |
| 白色 | `#FFFFFF` | 文字、对比 |
| 深灰 | `#1F2937` | 正文、副标题 |

### 设计风格

**整体风格**：简洁科技风

- **扁平化设计**：避免复杂渐变和阴影
- **留白充足**：信息层次清晰
- **图标化元素**：使用连接、节点、AI 相关图标
- **现代感**：无衬线字体，几何元素

### 视觉元素建议

1. **背景**
   - 纯色或微妙渐变（深蓝 → 浅蓝）
   - 可添加细微网格/电路板纹理

2. **核心图形**
   - 连接节点图（象征 MCP 连接概念）
   - AI 脑图/芯片图标
   - 简约的线条连接图案

3. **装饰元素**
   - 圆角矩形卡片
   - 光效/发光点
   - 简洁的图标（齿轮、链接、云端）
