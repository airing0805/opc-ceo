# 日志查看

## 日志位置

```
.claude/logs/
├── <taskName>.log          # 执行日志
└── <taskName>.error.log    # 错误日志
```

## 查看方式

### 1. 查看最新日志

```powershell
Get-Content .claude\logs\<taskName>.log -Tail 50
```

### 2. 查看错误日志

```powershell
Get-Content .claude\logs\<taskName>.error.log -Tail 20
```

### 3. 查看所有日志

```powershell
Get-ChildItem .claude\logs\*.log | ForEach-Object {
    Write-Host "=== $($_.Name) ===" -ForegroundColor Cyan
    Get-Content $_.FullName -Tail 20
    Write-Host ""
}
```

## 日志格式

```
[2026-02-25 12:00:00] Starting task...
[task output...]
[2026-02-25 12:05:00] Task completed.
```

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 日志文件不存在 | 任务可能未执行过 |
| 错误日志有内容 | 检查执行脚本和权限 |
| 日志过大 | 定期清理旧日志 |

## 日志清理建议

- 保留最近 7 天的日志
- 压缩归档历史日志
- 使用日志轮转
