## MCP 工具使用规范

| 操作类型 | 首选工具 | 备选工具 | 使用场景 |
|----------|----------|----------|----------|
| 文档读取 | `mcp__filesystem__read_file` | `Read` | 所有文档读取 |
| 文档写入 | `mcp__filesystem__write_file` | `Write` | 创建/覆盖文档 |
| 文档编辑 | `Edit` | `mcp__filesystem__edit_file` | 精确字符串替换 |
| 文件搜索 | `mcp__filesystem__search_files` | `Glob` | 按名称搜索 |
| 目录列表 | `mcp__filesystem__list_directory` | `Bash ls` | 查看目录内容 |
| 目录树 | `mcp__filesystem__directory_tree` | - | 查看目录结构 |

---

## 优先级规则

```
工具选择优先级（文档操作）：
1. MCP 工具 > 原生工具
2. Edit 工具仅用于精确编辑（字符串替换）
3. Bash 仅用于系统命令（git, npm, etc.）
```
