# 快速参考

## 常用命令

### PowerShell 命令

| 操作 | 命令 |
|------|------|
| 列出所有任务 | `Get-ScheduledTask` |
| 查看任务信息 | `Get-ScheduledTaskInfo -TaskName <名称>` |
| 手动运行任务 | `Start-ScheduledTask -TaskName <名称>` |
| 启用任务 | `Enable-ScheduledTask -TaskName <名称>` |
| 禁用任务 | `Enable-ScheduledTask -TaskName <名称>` |
| 删除任务 | `Unregister-ScheduledTask -TaskName <名称>` |

### 项目脚本

| 操作 | 脚本 |
|------|------|
| 注册任务 | `powershell .claude/schedules/register-task.ps1` |
| 删除任务 | `powershell .claude/schedules/unregister-task.ps1` |
| 查看日志 | `powershell .claude/schedules/view-logs.ps1` |

## Cron 表达式速查

| 表达式 | 说明 |
|--------|------|
| `* * * * *` | 每分钟 |
| `*/5 * * * *` | 每 5 分钟 |
| `*/15 * * * *` | 每 15 分钟 |
| `*/20 * * * *` | 每 20 分钟 |
| `*/30 * * * *` | 每 30 分钟 |
| `0 * * * *` | 每小时 |
| `0 */2 * * *` | 每 2 小时 |
| `0 9 * * *` | 每天 9:00 |
| `0 9 * * 1-5` | 工作日 9:00 |
| `0 0 * * 0` | 每周日午夜 |

## 文件结构
