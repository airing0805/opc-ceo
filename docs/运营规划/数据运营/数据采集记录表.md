## 一、采集说明

### 1.1 采集频率

| 频率 | 时间 | 采集范围 |
|------|------|----------|
| 每日 | 每晚 21:00 | 当日新发布内容（前7天） |
| 每周 | 周日晚 21:00 | 所有追踪中的内容 |
| 月度 | 每月1日 | 全量数据汇总 |

### 1.2 采集入口

| 平台 | 数据页面 | 登录要求 |
|------|----------|----------|
| 知乎 | https://www.zhihu.com/creator/content-management/all | 登录1次 |
| 简书 | https://www.jianshu.com/writer#/stats | 登录1次 |
| CSDN | https://mp.csdn.net/mp_blog/creation/success | 登录1次 |
| 掘金 | https://juejin.cn/creator/content | 登录1次 |

### 1.3 采集字段

| 字段 | 说明 | 单位 |
|------|------|------|
| 阅读量 | 文章被阅读次数 | 次 |
| 点赞数 | 获得点赞数量 | 个 |
| 评论数 | 获得评论数量 | 条 |
| 收藏数 | 被收藏次数 | 次 |
| 分享数 | 被分享次数 | 次 |
| 转发数 | 被转发次数 | 次 |
