## 附录：模板使用说明

### 必填 vs 选填

| 部分 | 必填性 | 说明 |
|------|--------|------|
| 日志头部 | **必填** | 基本信息 |
| 执行目标 | **必填** | 明确目标 |
| 执行结果 | **必填** | 结果统计 |
| 问题与决策 | 选填 | 有问题/决策时填 |
| 后续行动 | 选填 | 有待办时填 |
| 经验总结 | 选填 | 有经验时填 |
| 相关文档 | 选填 | 有关联文档时填 |

### 状态标记规范

| 状态 | 符号 | 说明 |
|------|------|------|
| 全部完成 | ✅ | 所有目标达成 |
| 部分完成 | ⚠️ | 部分目标达成 |
| 未完成 | ❌ | 主要目标未达成 |
| 进行中 | 🔄 | 执行尚未结束 |
| 已暂停 | ⏸️ | 等待条件恢复 |

### 日志命名规范

格式：`EXEC-YYYY-MM-DD-NNN`

示例：`EXEC-2026-02-27-001`

- EXEC：固定前缀
- YYYY-MM-DD：执行日期
- NNN：当日序号（001, 002, 003...）
