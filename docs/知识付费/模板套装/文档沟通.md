# 文档沟通规范

## 沟通文档类型

| 类型 | 文件名 | 用途 |
|------|--------|------|
| 任务分配 | 任务分配.md | 任务分配记录 |
| 汇报总结 | 汇报总结.md | 工作汇报 |
| 协调记录 | 协调记录.md | 跨角色协调 |
| 问题清单 | 问题清单.md | 问题追踪 |
| 决策记录 | 决策记录.md | 重要决策记录 |

## 文档格式规范

### 任务分配格式

```markdown
### 分配给 {角色名称}

**任务 ID**: {TASK-YYYY-MM-DD-NNN}
**分配时间**: {YYYY-MM-DD HH:MM}
**优先级**: {P1/P2/P3/P4}
**负责人**: {角色名称}
**任务描述**: {描述内容}
**验收标准**: {验收条件}
**执行状态**: {待执行/进行中/已完成}
**截止日期**: {YYYY-MM-DD}
**关联任务**: {相关任务ID}
```

### 汇报格式

```markdown
### {角色名称} 汇报 - {日期}

**汇报类型**: {日报/周报/月报}
**汇报时间**: {YYYY-MM-DD}

#### 完成事项
- {事项1}
- {事项2}

#### 进行中事项
- {事项1} - 进度 {XX}%

#### 遇到的问题
- {问题描述} - {处理方案}

#### 下一步计划
- {计划1}
- {计划2}
```

### 协调记录格式

```markdown
### 协调记录 - {COORD-YYYY-MM-DD-NNN}

**协调时间**: {YYYY-MM-DD HH:MM}
**发起方**: {角色A}
**参与方**: {角色B}, {角色C}
**协调类型**: {任务协作/资源调配/问题解决}

#### 协调事项
{协调事项描述}

#### 讨论内容
- {要点1}
- {要点2}

#### 结论
{协调结论}

#### 后续行动
| 行动项 | 负责人 | 完成时间 |
|--------|--------|----------|
| {行动1} | {角色} | {时间} |
| {行动2} | {角色} | {时间} |
```

### 问题清单格式

```markdown
### 问题记录 - {PROB-YYYY-MM-DD-NNN}

**发现时间**: {YYYY-MM-DD HH:MM}
**发现人**: {角色名称}
**严重程度**: {P1/P2/P3/P4}
**状态**: {待处理/处理中/已解决/已关闭}

#### 问题描述
{问题详细描述}

#### 影响范围
{受影响的系统/功能/用户}

#### 解决方案
{处理方案描述}

#### 处理记录
| 时间 | 处理人 | 操作 | 结果 |
|------|--------|------|------|
| {时间} | {角色} | {操作} | {结果} |
```

## 沟通流程

### 任务分配流程
```
1. 发起方创建任务分配文档
2. 明确任务内容、优先级、截止日期
3. 指定负责人
4. 负责人确认接收
5. 开始执行
```

### 汇报流程
```
1. 按周期（日报/周报/月报）生成汇报
2. 汇报内容包含：完成事项、进行中事项、问题、计划
3. 提交给上级/相关角色
4. 归档到沟通文档目录
```

### 协调流程
```
1. 发起协调请求
2. 相关方参与讨论
3. 达成共识
4. 记录结论和后续行动
5. 跟踪执行情况
```
