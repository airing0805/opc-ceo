# 知识存储

## 功能概述

使用 MCP Memory 存储知识实体（笔记、概念、项目、资源、人物）。

## 操作命令

### 命令 1：存储笔记

**用途**：创建笔记实体

**语法**：
```
存储笔记:
  - title: <笔记标题>
  - content: <笔记内容>
  - tags: <标签> (可选)
  - category: <分类> (可选)
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| title | string | 是 | 笔记标题 |
| content | string | 是 | 笔记内容 |
| tags | string | 否 | 标签，逗号分隔 |
| category | string | 否 | 分类 |

**示例**：
```
存储笔记:
  - title: MVP 设计思路
  - content: 使用最小可行产品的方法验证核心假设...
  - tags: 设计,MVP
  - category: 设计思路
```

---

### 命令 2：存储概念

**用途**：创建概念/想法实体

**语法**：
```

  - name: <概念名称>
  - description: <描述>
  - type: <类型> (可选)
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |

| name | string | 是 | 概念名称 |
| description | string | 是 | 概念描述 |
| type | string | 否 | 概念类型：设计模式/算法/工具等 |

**示例**：
```
存储概念:
  - name: 单例模式

  - type: 设计模式
```

## MCP Memory 使用

### 创建笔记实体

```
使用 mcp__memory__create_entities:
{
  "entities": [{
    "name": "NOTE-2026-02-23-001: MVP 设计思路",
    "entityType": "Note",
    "observations": [
      "title: MVP 设计思路",
      "content: 使用最小可行产品的方法验证核心假设...",
      "tags: 设计,MVP",
      "category: 设计思路",
      "createdAt: 2026-02-23T17:00:00"
    ]
  }]
}
```

### 创建概念实体

```
使用 mcp__memory__create_entities:
{
  "entities": [{
    "name": "CONCEPT-单例模式",
    "entityType": "Concept",
    "observations": [
      "name: 单例模式",
      "description: 确保一个类只有一个实例",
      "type: 设计模式",
      "createdAt: 2026-02-23T17:00:00"
    ]
  }]
}
```

### 创建关系

```
使用 mcp__memory__create_relations:
{
  "relations": [{
    "from": "CONCEPT-单例模式",
    "to": "PROJ-001: OPC-CEO",
    "relationType": "used_in"
  }]
}
```

---

## 实体 ID 格式

| 类型 | 格式 | 示例 |
|------|------|------|
| Note | `NOTE-YYYY-MM-DD-NNN` | `NOTE-2026-02-23-001` |
| Concept | `CONCEPT-<名称>` | `CONCEPT-单例模式` |
| Project | `PROJ-NNN` | `PROJ-001` |
| Resource | `RESOURCE-<类型>-NNN` | `RESOURCE-book-001` |
| Person | `PERSON-<名称>` | `PERSON-Martin Fowler` |
