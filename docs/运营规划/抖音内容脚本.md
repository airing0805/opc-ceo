## 一、脚本使用说明

### 1.1 脚本结构

每期脚本包含以下模块：

| 模块 | 时长 | 作用 |
|------|------|------|
| 开场hook | 3秒 | 抓住注意力，触发完播 |
| 核心内容 | 30-45秒 | 传递核心价值 |
| 结尾引导 | 5-10秒 | 引导互动、关注 |

### 1.2 拍摄建议

| 项目 | 建议 |
|------|------|
| 环境 | 安静、光线充足、背景简洁 |
| 设备 | 手机横屏/竖屏均可，建议竖屏 |
| 服装 | 简约商务，避免花哨 |
| 表情 | 自然自信，适度微笑 |
| 语速 | 每秒2-3字，留白停顿 |


## 四、第03期脚本

### 4.1 基本信息

| 项目 | 内容 |
|------|------|
| 标题 | 5个AI工具，一个人就是一家公司 |
| 形式 | 混剪演示 |
| 时长 | 40秒 |
| 核心卖点 | 工具展示、效率提升、实用推荐 |

### 4.2 脚本结构

```
【0-5秒】效果展示
画面：AI生成文章、数据报表、设计图的快速切换
配音："这5个AI工具，让我一个人干了10个人的活"

【5-20秒】工具演示
画面1：Claude写作演示
配音："写作用Claude，帮我写篇带货文案，3秒出初稿"

画面2：Midjourney作图演示
配音："作图用Midjourney，描述你想要的效果，10秒出图"

画面3：Notion管理演示
配音："管理用Notion，一个页面管项目、管客户、管内容"

【20-35秒】效率对比
画面：传统方式vs AI方式的时间对比
配音："以前3小时的工作，现在30分钟完成"

【35-40秒】引导关注
画面：工具界面+关注引导
配音："这5个工具怎么用，下期详细讲，点关注不迷路"
```

### 4.3 素材需求

| 素材 | 说明 |
|------|------|
| Claude界面录屏 | 写作演示，约5秒 |
| Midjourney界面录屏 | 作图演示，约5秒 |
| Notion界面录屏 | 管理演示，约5秒 |
| 背景音乐 | 节奏感强，抖音热歌 |
| 字幕 | 关键信息放大显示 |

### 4.4 备注

- 混剪节奏要快，每2-3秒切换一个画面
- 背景音乐选择卡点节奏
- 关键文字用字幕放大
