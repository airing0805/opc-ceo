# 自我进化

## 进化机制

### 触发条件

| 触发类型 | 条件 | 频率 |
|----------|------|------|
| 定时触发 | 每日自动执行 | 24h |
| 事件触发 | 任务失败/异常 | 实时 |
| 手动触发 | 用户请求 | 按需 |

### 进化流程

```
1. 状态检查 → 2. 问题识别 → 3. 方案生成 → 4. 执行优化 → 5. 效果验证
```

## 自我诊断清单

### 日常检查
- [ ] Memory 连接状态正常
- [ ] Git 仓库状态正常
- [ ] 近期任务完成率达标
- [ ] 无重大错误日志

### 周期性检查
- [ ] 配置参数是否最优
- [ ] 技能是否需要更新
- [ ] 文档是否需要同步
- [ ] 协作流程是否顺畅

### 深度检查
- [ ] 性能指标分析
- [ ] 用户反馈收集
- [ ] 竞品功能对比
- [ ] 技术栈更新评估
