# 任务删除

## 删除流程

```
用户请求 → 确认任务 → 执行删除 → 更新 Memory → 清理文件
```

## 删除步骤

1. 确认任务名称
2. 执行注销脚本
3. 更新 MCP Memory
4. 清理相关文件（可选）

## PowerShell 命令

```powershell
# 删除任务
Unregister-ScheduledTask -TaskName <TaskName> -Confirm:$false
```

## 注销脚本模板

```powershell
$taskName = "<TaskName>"

Unregister-ScheduledTask -TaskName $taskName -Confirm:$false

Write-Host "Task deleted: $taskName" -ForegroundColor Green
```

## Memory 更新

更新 Task 实体状态：

```
实体名称: TASK-YYYY-MM-DD-NNN
更新状态: deleted
```

## 文件清理（可选）

删除相关文件：
- 执行脚本: `.claude/schedules/<taskName>.bat`
- 注册脚本: `.claude/schedules/register-<taskName>.ps1`

**警告**：日志文件建议保留，用于历史记录。
