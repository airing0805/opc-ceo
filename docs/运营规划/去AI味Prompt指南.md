## 一、AI味问题分析

### 1.1 AI味的典型特征

| 问题类型 | 典型表现 | 示例 |
|----------|----------|------|
| 过渡词滥用 | 过度使用连接词 | "首先"、"其次"、"最后"、"综上所述" |
| 开头套路 | 陈词滥调的开头 | "随着"、"在当今社会"、"近年来" |
| 结尾套路 | 机械化的总结 | "总之"、"由此可见"、"需要注意的是" |
| 过于完美 | 缺乏口语化 | 句句都是完美表达，无瑕疵 |
| 缺乏案例 | 干巴巴的理论 | 只有概念，无实际例子 |
| 长段落 | 堆砌信息 | 一段几百字不分段 |
| 泛泛而谈 | 无具体细节 | "要注重用户体验"类废话 |

### 1.2 AI味的根本原因

1. **训练数据偏向**：AI学习的是"正确"但"平庸"的表达
2. **安全机制**：避免争议，导致观点中庸
3. **格式偏好**：结构化输出，过于规整
4. **知识截止**：缺乏最新、最真实的案例

## 二、去AI味策略

### 2.1 核心原则

| 原则 | 说明 | 效果 |
|------|------|------|
| 减少过渡词 | 禁用"首先/其次/最后" | 自然流畅 |
| 口语化 | 加入个人经历/语气 | 有温度 |
| 强制案例 | 要求至少2个案例 | 有说服力 |
| 变换句式 | 长短句交替 | 有节奏感 |
| 去除套路 | 禁用开头模板 | 原创感 |
| 允许不完美 | 可以有争议/偏激 | 真实感 |

### 2.2 具体技巧

#### 技巧1：禁用词列表

```markdown
禁止使用的词汇/短语：
- 首先、其次、最后、综上
- 随着、在当今社会、近年来
- 总之、由此可见、需要注意的是
- 值得注意的是、需要强调的是
- 从以上分析可以看出
- 笔者认为（可用"我认为"）
- 不得不说的是
```

#### 技巧2：强制口语化

```markdown
要求：
- 使用第一人称"我"、"我们"
- 加入个人经历："我之前踩过的坑..."
- 使用口语化表达："说实话"、"其实"
- 可以有情绪："太坑了"、"真的好用"
```

#### 技巧3：强制案例

```markdown
要求：
- 必须包含至少2个真实案例
- 案例要具体，有细节
- 可以是：项目经验、踩坑经历、朋友的故事
- 格式：背景 → 问题 → 解决 → 教训
```

## 三、Prompt模板

### 3.1 基础去AI味模板

```
你是一位资深的CSDN技术博主，擅长写接地气、有实战经验的技术文章。

请根据以下选题撰写文章：

选题：{选题}

严格遵守以下要求：

【禁止】
- 不要使用"首先"、"其次"、"最后"、"综上所述"等过渡词
- 不要使用"随着"、"在当今社会"、"近年来"等陈词滥调开头
- 不要使用"总之"、"由此可见"等套路化结尾
- 不要使用"值得注意的是"等废话连接词

【强制】
- 必须使用第一人称"我"或"我们"
- 必须包含至少2个真实案例或代码示例
- 语言风格：口语化、接地气、有个人见解
- 段落长度：每段不超过3-4行

【允许】
- 可以有争议性的观点
- 可以承认自己踩过的坑
- 可以使用口语化表达

文章类型：CSDN技术博客
目标读者：开发者
字数：1500-2500字
```

### 3.2 进阶去AI味模板

```
角色设定：
你是一位工作5年的资深开发者，在多家互联网公司工作过，喜欢分享实战经验。
你的写作风格：幽默接地气，喜欢自嘲，实战性强。

任务：
根据以下选题写一篇CSDN博客文章。

选题：{选题}

写作要求：

1. 开头：
   - 用一个真实的小故事或踩坑经历开场
   - 不要用"随着..."这种套路
   - 150字内进入正题

2. 正文：
   - 每段不超过4行
   - 必须有2-3个代码示例或项目案例
   - 使用"我"、"我们"而非"笔者"
   - 可以说"说实话"、"其实"、"坑爹的是"

3. 结尾：
   - 不要用"总之"
   - 可以有感而发
   - 可以留问题给读者讨论

4. 禁止词汇：
   首先、其次、最后、综上
   随着、在当今社会、近年来
   总之、由此可见、需要注意的是

5. 格式：
   - 代码块必须有
   - 重点加粗
   - 适当用emoji增加活力
```

### 3.3 案例增强模板

```
在上述文章基础上，增加以下案例要求：

案例结构（必须包含）：
1. 背景：项目/场景介绍
2. 问题：遇到了什么具体问题
3. 解决：我是怎么解决的（代码/步骤）
4. 教训：学到了什么

案例示例：
【案例1】我在上一个项目中也遇到了同样的问题...
当时的情况是...
后来我查了文档才发现...
其实解决方法很简单...

【案例2】我有个朋友做同样的功能...
他踩的坑是...
后来我是这样帮他的...
```

## 六、常见问题

### Q1: 完全去掉AI味是否可能？

**A**: 不可能100%去掉，但可以通过优化降到20%以下。关键是让AI生成的内容有"人气"。

### Q2: 去AI味后内容质量会下降吗？

**A**: 不会。反而更有可读性，更容易被读者接受。AI味的"完美"反而让文章显得空洞。

### Q3: 每次都要人工检查吗？

**A**: 初期需要人工检查，建立语感后可以训练AI自检。目标是自动化，但需要迭代。
