# 财务报告

## 概述

生成各种财务报告，包括日报、周报、月报，支持按分类、项目等维度汇总分析。

## 报告类型

### 日报

**内容**：
- 当日收入总额
- 当日支出总额
- 当日净收入
- 交易明细

**生成方式**：

```
生成 2026-02-23 日报
→ 查询日期为 2026-02-23 的所有交易
→ 按收入/支出汇总
→ 生成日报
```

### 周报

**内容**：
- 本周收入总额
- 本周支出总额
- 本周净收入
- 按分类汇总
- 与上周对比

**生成方式**：

```
生成 2026-02-17 至 2026-02-23 周报
→ 查询日期范围内的所有交易
→ 按分类汇总
→ 生成周报
```

### 月报

**内容**：
- 本月收入总额
- 本月支出总额
- 本月净收入
- 按分类汇总
- 与上月对比
- 趋势分析

**生成方式**：

```
生成 2026-02 月报
→ 查询 2026-02 所有交易
→ 按分类汇总
→ 生成月报
```

## 报告格式

### 日报格式

```
财务日报 - 2026-02-23
========================

【汇总】
收入: ¥5,000.00
支出: ¥299.00
净收入: ¥4,701.00

【明细】
1. TXN-2026-02-23-001 | 收入 | ¥5,000.00 | 项目收入 | 某项目开发收入
2. TXN-2026-02-23-002 | 支出 | ¥299.00 | 软件订阅 | 一年软件订阅
```

### 月报格式

```
财务月报 - 2026年2月
========================

【汇总】
总收入: ¥15,000.00
总支出: ¥2,500.00
净收入: ¥12,500.00

【收入分类】
项目收入: ¥10,000.00 (66.7%)
咨询服务: ¥3,000.00 (20.0%)
其他收入: ¥2,000.00 (13.3%)

【支出分类】
软件订阅: ¥1,200.00 (48.0%)
云服务: ¥800.00 (32.0%)
学习培训: ¥500.00 (20.0%)

【趋势】
较上月: +15.3%
较去年同月: +8.7%
```

## 高级分析

### 项目收支分析

```
查询 PROJ-001 项目收支
→ 汇总收入
→ 汇总支出
→ 计算利润率
```

### 分类趋势分析

```
分析软件订阅支出趋势
→ 按月汇总 EXP-SUB
→ 生成趋势图
→ 预测下月支出
```

## 报告导出

### 支持格式

| 格式 | 说明 |
|------|------|
| 文本 | Markdown 格式 |
| CSV | 表格数据导出 |
| JSON | 结构化数据导出 |

### 导出方式

```
导出为 CSV
→ 生成 CSV 文件
→ 保存到指定位置
```

## MCP Memory 存储

### 报告实体

```yaml
entityType: FinancialReport
name: REPORT-2026-02
observations:
  - type: monthly
  - period: 2026-02
  - totalIncome: 15000
  - totalExpense: 2500
  - netIncome: 12500
  - incomeBreakdown:
    - category: INC-PROJ
      amount: 10000
      percentage: 66.7
  - expenseBreakdown:
    - category: EXP-SUB
      amount: 1200
      percentage: 48.0
  - createdAt: 2026-02-23T10:00:00Z
```

## 注意事项

- 金额保留 2 位小数
- 百分比四舍五入到 1 位小数
- 趋势对比需有基准数据
