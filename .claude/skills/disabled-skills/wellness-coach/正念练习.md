# 正念练习

## 功能概述

提供各种正念练习的引导和指导，帮助用户缓解压力、改善情绪、提升专注力和整体心理健康。

## 练习类型

### 深呼吸练习（2-5分钟）

**适用场景**：紧张、焦虑、需要快速平静时

**步骤**：
1. 找一个舒适的坐姿或站姿
2. 闭上眼睛或轻柔注视一点
3. 通过鼻子缓慢吸气，心中默数4下
4. 屏住呼吸，心中默数2下
5. 通过嘴缓慢呼气，心中默数6下
6. 重复这个过程2-5分钟

**引导语示例**：
```
"现在，让我们开始深呼吸练习。
感受空气从鼻子进入，填满你的肺部。
屏住呼吸，感受空气在体内的停留。
然后，缓慢地将空气从嘴里呼出，感受身体放松下来。
继续这样的呼吸节奏，让每一次呼吸都带走一些紧张。"
```

### 身体扫描（10-15分钟）

**适用场景**：休息放松、睡前、身体疲劳时

**步骤**：
1. 躺下或舒适坐下
2. 闭上眼睛，从头到脚逐一关注身体各部位
3. 每个部位停留10-30秒，感受当下的感觉
4. 不加判断地观察，无论是紧张、舒适还是麻木
5. 结束时感受全身的状态

**顺序**：
```
额头 → 眉毛 → 眼睛 → 鼻子 → 嘴巴 → 下巴
脸颊 → 耳朵 → 颈部 → 肩膀 → 手臂 → 手指
胸部 → 腹部 → 背部 → 臀部 → 大腿 → 小腿 → 脚趾
```

### 正念行走（15-20分钟）

**适用场景**：午间休息、需要换环境时

**步骤**：
1. 选择一个安静的小路或室内空间来回行走
2. 放慢脚步，每一步都仔细感受
3. 注意脚与地面的接触
4. 感受身体移动时的平衡和节奏
5. 当注意力分散时，温柔地回到脚步的感觉上

**引导语示例**：
```
"让我们开始正念行走。
感受左脚抬起，移动，然后接触地面。
感受重心的转移。
然后是右脚抬起，移动，接触地面。
保持这样的节奏，每一步都是一个新的开始。"
```

### 冥想引导（20-30分钟）

**适用场景**：晨起、睡前、深度放松时

**步骤**：
1. 找一个舒适的坐姿（椅子或地面）
2. 背部挺直但放松
3. 将注意力集中在呼吸上
4. 当思绪出现时，观察它们而不跟随
5. 温柔地将注意力带回到呼吸上
6. 结束时慢慢张开眼睛，伸展身体

**引导语示例**：
```
"现在，让我们进入冥想时间。
找到一个舒适的坐姿，让身体放松下来。
将注意力带回到呼吸上，感受气息的流动。
当思绪飘走时，不要责怪自己，只是观察它，然后回到呼吸。
这就是冥想的全部：一次又一次地回到当下。"
```

### 感恩练习（5-10分钟）

**适用场景**：睡前、情绪低落、需要正面心态时

**步骤**：
1. 回顾今天发生的三件好事（可以是小事）
2. 对每一件事写下原因
3. 感受内心的感激之情
4. 可以将感谢写在感恩日记中

**示例**：
```
1. 今天完成了重要任务 - 感谢自己的努力和能力
2. 和朋友聊了天 - 感谢有这样支持我的人
3. 天气很好 - 感谢大自然的馈赠
```

## 练习选择指南

| 可用时间 | 心理状态 | 推荐练习 | 时长 |
|----------|----------|----------|------|
| 2-5分钟 | 紧张、焦虑 | 深呼吸 | 2-5分钟 |
| 10-15分钟 | 需要放松 | 身体扫描 | 10-15分钟 |
| 15-20分钟 | 需要换环境 | 正念行走 | 15-20分钟 |
| 20-30分钟 | 需要深度放松 | 冥想引导 | 20-30分钟 |
| 5-10分钟 | 情绪低落 | 感恩练习 | 5-10分钟 |

## 最佳实践

1. **从短时间开始** - 新手从2-5分钟练习开始，逐渐增加
2. **定期练习** - 每天练习比偶尔长时间练习更有效
3. **不判断自己** - 思绪分散是正常的，只是温柔地回来
4. **创造仪式感** - 固定的时间和地点有助于形成习惯
5. **记录效果** - 记录每种练习对自己最有效

## 注意事项

- 正念练习不等于什么都不想，而是觉察当下
- 初学者思绪分散是正常的，不要责怪自己
- 如果感到不适，可以随时停止或调整方式
- 孕期、严重心脏病等人群请在医生指导下练习
- 正念练习不能替代专业心理治疗

## 高级技巧

### 呼吸计数法

在冥想中，每呼气一次计数一次，数到10再回到1，帮助保持专注。

### 标记思维法

当思维出现时，轻柔地标记它（如"思考"、"记忆"、"计划"），然后回到呼吸。

### 五感练习

在日常生活中练习正念，将注意力依次放在：
1. 看到5样东西
2. 听到4种声音
3. 触摸到3种质感
4. 嗅到2种气味
5. 尝到1种味道

## 记录与追踪

虽然正念练习本身不需要记录，但可以追踪练习习惯：

```
可以记录在 WellnessRecord 中：
- 今日练习了哪种正念方法
- 练习时长的感觉如何
- 练习后的状态变化
```

这有助于发现哪种练习最有效，以及坚持的情况。
