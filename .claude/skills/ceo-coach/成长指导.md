# 成长指导

## 动态执行规范

**指导前必须**：
1. 从 Memory 读取评估结果
2. 从 Memory 读取当前标准
3. 分析成长差距
4. 制定指导方案

---

## 成长触发条件

### 自动触发

| 触发条件 | 阈值 | 触发动作 |
|----------|------|----------|
| 决策失误 | 连续2次 | 立即分析 + 指导 |
| 效率下降 | >20% | 原因分析 + 优化 |
| 评估不达标 | 任意维度 | 针对性指导 |
| 阶段结束 | 时间到 | 阶段评估 |

### 手动触发

| 触发条件 | 说明 |
|----------|------|
| 用户反馈 | 用户表达不满或建议 |
| 新需求 | 现有能力无法满足 |
| 主动请求 | CEO/Coach 请求帮助 |

---

## 指导类型

| 类型 | 场景 | 方式 |
|------|------|------|
| 即时指导 | 紧急问题 | 立即反馈 |
| 周期指导 | 常规改进 | 周报包含 |
| 专项指导 | 能力短板 | 专项训练 |
| 阶段指导 | 阶段进阶 | 系统提升 |

---

## 指导流程

```
┌────────────┐    ┌────────────┐    ┌────────────┐
│ 1.读取评估  │───▶│ 2.分析差距  │───▶│ 3.制定方案  │
└────────────┘    └────────────┘    └────────────┘
                                          │
                                          ▼
┌────────────┐    ┌────────────┐    ┌────────────┐
│ 6.验证效果  │◀───│ 5.跟踪执行  │◀───│ 4.实施指导  │
└────────────┘    └────────────┘    └────────────┘
```

### 步骤详解

**1. 读取评估**
- 从 Memory 读取最近评估结果
- 识别不达标的维度

**2. 分析差距**
- 对比当前水平与标准
- 识别根本原因

**3. 制定方案**
- 设定改进目标
- 制定具体措施

**4. 实施指导**
- 执行改进措施
- 记录执行过程

**5. 跟踪执行**
- 监控改进进度
- 调整方案

**6. 验证效果**
- 评估改进效果
- 更新 Memory

## 指导输出格式

```markdown
# 成长指导

**触发原因**: <从评估/反馈读取>
**问题领域**: <从分析得出>
**指导类型**: 即时/周期/专项/阶段

## 问题分析

- 问题描述: <具体描述>
- 根本原因: <分析结果>
- 影响范围: <影响评估>

## 改进方案

- 改进目标: <SMART目标>
- 具体措施:
  1. <措施1>
  2. <措施2>
- 预期效果: <预期结果>

## 跟踪计划

- 检查点1: <日期> - <检查内容>
- 检查点2: <日期> - <检查内容>
- 最终验证: <日期> - <验证方式>
```

---

## 效果验证

### 验证维度

| 维度 | 验证方式 | 达标标准 |
|------|----------|----------|
| 问题解决 | 重新评估 | 恢复到合格线以上 |
| 能力提升 | 对比前后数据 | 提升幅度达标 |
| 持续稳定 | 持续观察 | 连续3次无回退 |

### 验证输出

```
验证结果:
- 指导ID: <ID>
- 验证时间: <时间>
- 改进前: <指标>
- 改进后: <指标>
- 效果评估: 达标/部分/未达标
- 后续行动: <如有>
```

## Memory 使用

### 创建指导记录

```
mcp__memory__create_entities:
{
  "entities": [{
    "name": "coach-guide-YYYY-MM-DD-NNN",
    "entityType": "Coaching",
    "observations": [
      "date: YYYY-MM-DD",
      "trigger: 触发原因",
      "area: 问题领域",
      "type: 即时/周期/专项/阶段",
      "issue: 问题描述",
      "cause: 根本原因",
      "goal: 改进目标",
      "actions: [<具体措施>]",
      "status: 进行中/已完成",
      "createdAt: YYYY-MM-DDTHH:MM:SS"
    ]
  }]
}
```

### 更新验证结果

```
mcp__memory__add_observations:
{
  "observations": [{
    "entityName": "coach-guide-YYYY-MM-DD-NNN",
    "contents": [
      "verifiedAt: YYYY-MM-DD",
      "beforeValue: <改进前>",
      "afterValue: <改进后>",
      "result: 达标/部分/未达标",
      "followUp: <后续行动>"
    ]
  }]
}
```

### 查询指导历史

```
mcp__memory__search_nodes:
query: "Coaching"
```
