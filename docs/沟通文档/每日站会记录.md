## 使用说明

1. **每日站会时间**：每天工作开始前5分钟
2. **站会内容**：

   - 阻塞：今天遇到什么问题

3. **阻塞处理**：阻塞问题必须在24小时内解决或升级
4. **记录归档**：每周末整理本周站会记录到周报
