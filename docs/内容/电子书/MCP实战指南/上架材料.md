## 产品描述

MCP（Model Context Protocol）是 Anthropic 推出的开放标准，让 AI 助手能够安全、高效地连接外部数据源和工具。

本电子书从零开始，带你全面掌握 MCP：
- **第一章**：理解 MCP 是什么，为什么需要它
- **第二章**：手把手安装配置 Claude Desktop 和 Claude Code
- **第三章**：4个实战案例，即学即用
- **第四章**：开发自己的 MCP 服务器，进阶技巧

适合人群：
- 想让 AI 助手访问本地文件、数据库、API 的开发者
- 想用 AI 自动化工作流的效率追求者
- 想开发 AI 原生应用的产品经理
