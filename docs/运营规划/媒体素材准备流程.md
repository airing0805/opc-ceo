## 一、素材需求概述

### 1.1 素材类型

| 类型 | 用途 | 频率 | 优先级 |
|------|------|------|--------|
| 文章配图 | 文章内容插图 | 每篇 2-5 张 | P0 |
| 封面图 | 文章/专栏封面 | 每篇 1 张 | P1 |
| 头像/Logo | 账号基础设置 | 一次性 | P0 |
| 图标素材 | 文章装饰 | 按需 | P2 |

### 1.2 素材来源分布

| 来源 | 占比 | 适用场景 |
|------|------|----------|
| **AI 生成** | 60% | 文章配图、封面图、示意图 |
| **互联网查找** | 30% | 截图、免费图库、官方素材 |
| **用户提供** | 10% | 特殊需求、品牌素材 |

## 二、AI 素材生成流程

### 2.1 文章配图生成

```
┌─────────────────────────────────────────────────────────┐
│ Step 1: 分析文章内容，确定配图需求                         │
│   - 识别关键概念                                         │
│   - 确定配图位置                                         │
│   - 定义配图风格                                         │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ Step 2: 生成 Prompt                                      │
│   - 描述图片内容                                         │
│   - 指定风格（扁平化/插画/照片感）                         │
│   - 设置尺寸比例                                         │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ Step 3: 调用 AI 模型生成                                  │
│   - 使用用户指定的模型                                   │
│   - 生成 2-3 个候选                                      │
│   - 选择最佳结果                                         │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ Step 4: 后处理（如需要）                                  │
│   - 裁剪调整尺寸                                         │
│   - 添加文字（可选）                                     │
│   - 压缩优化                                             │
└─────────────────────────────────────────────────────────┘
```

### 2.2 Prompt 模板

#### 技术教程配图

```
风格: 扁平化插画 / 科技感
内容: [描述具体内容]
颜色: 蓝色系 / 渐变色
尺寸: 16:9 或 4:3
避免: 人物肖像、版权内容
```

#### 概念示意图

```
风格: 信息图 / 流程图
内容: [描述流程或结构]
形式: 图标 + 连线 + 标签
尺寸: 1:1 或 16:9
```

## 三、互联网素材查找

### 3.1 免费图库推荐

| 图库 | 网址 | 特点 | 适用场景 |
|------|------|------|----------|
| Unsplash | unsplash.com | 高质量照片 | 背景图、封面 |
| Pexels | pexels.com | 免费 + 中文 | 通用配图 |
| Pixabay | pixabay.com | 多类型素材 | 插图、图标 |
| Iconfont | iconfont.cn | 中文图标 | 图标素材 |

### 3.2 查找规范

| 规范 | 说明 |
|------|------|
| **版权检查** | 仅使用 CC0 或可商用素材 |
| **来源记录** | 记录图片来源 URL |
| **尺寸要求** | 宽度 ≥800px |
| **格式要求** | 优先 JPG/PNG |

### 3.3 截图规范

| 场景 | 工具 | 要求 |
|------|------|------|
| 软件界面 | 系统截图 | 全屏后裁剪 |
| 代码截图 | Carbon / Ray.so | 使用代码美化工具 |
| 网页截图 | 浏览器开发者工具 | 隐藏敏感信息 |

## 四、AI 模型需求清单

### 4.1 需要用户提供的模型

| 序号 | 用途 | 推荐模型 | 用户选择 | 优先级 |
|------|------|----------|----------|--------|
| 1 | **文章配图生成** | Midjourney / DALL-E 3 / Stable Diffusion | 待填写 | P0 |
| 2 | **封面图设计** | Canva AI / 飞书设计 / 创客贴 | 待填写 | P1 |
| 3 | **图标素材** | Iconfont / Flaticon / AI生成 | 待填写 | P2 |
| 4 | **Logo/头像设计** | Looka / Hatchful / AI生成 | 待填写 | P0 |

### 4.2 模型选择说明

#### Midjourney
- **优点**: 质量高，风格多样
- **缺点**: 需要订阅，需通过 Discord
- **适用**: 高质量配图、封面图

#### DALL-E 3
- **优点**: 集成 ChatGPT，使用方便
- **缺点**: 需要订阅 ChatGPT Plus
- **适用**: 通用配图

#### Stable Diffusion
- **优点**: 免费开源，可本地部署
- **缺点**: 需要技术能力，硬件要求高
- **适用**: 批量生成

## 五、素材管理

### 5.1 素材存储结构

```
assets/
├── images/
│   ├── articles/          # 文章配图
│   │   ├── 2026-02/       # 按月份组织
│   │   │   ├── 001-*.png
│   │   │   └── 002-*.png
│   │   └── ...
│   ├── covers/            # 封面图
│   ├── icons/             # 图标
│   └── logos/             # Logo/头像
└── templates/             # 模板素材
```

### 5.2 命名规范

| 类型 | 命名格式 | 示例 |
|------|----------|------|
| 文章配图 | `{日期}-{序号}-{描述}.png` | `20260226-001-claude-code-tutorial.png` |
| 封面图 | `cover-{日期}-{标题}.png` | `cover-20260226-ai-tools.png` |
| 图标 | `icon-{名称}.svg` | `icon-github.svg` |

## 六、执行流程

### 6.1 每日素材准备

| 时间 | 任务 | 产出 |
|------|------|------|
| 内容发布前1小时 | 分析配图需求 | 配图清单 |
| 内容发布前30分钟 | 生成/查找素材 | 2-5张配图 |
| 内容发布前10分钟 | 插入文章并调整 | 完成配图 |

### 6.2 批量素材生成

| 周期 | 任务 | 产出 |
|------|------|------|
| 每周日 | 下周内容配图规划 | 配图需求清单 |
| 每周一 | 批量生成配图 | 10-20张配图 |
| 每周五 | 补充调整 | 补充素材 |
