# 数据看板设计文档

> 内容运营数据看板设计方案
> 版本：V1.0 | 创建日期：2026-02-27

---

## 一、看板块计概述

### 1.1 目的

设计统一的数据看板，实现内容运营数据的可视化展示，支持日常数据监控和决策分析。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 数据一致 | 统一数据源和计算口径 |
| 层级分明 | 区分汇总层和明细层 |
| 实时性 | 每日更新，支持趋势分析 |
| 可操作 | 数据可直接指导运营决策 |

## 三、仪表盘设计

### 3.1 核心指标卡片

**位置**：首页顶部，4个核心卡片横向排列

| 卡片 | 指标 | 展示内容 | 颜色编码 |
|------|------|----------|----------|
| 1 | 今日阅读 | 阅读量 + 环比变化 | 蓝色 |
| 2 | 今日互动 | 互动总数 + 互动率 | 绿色 |
| 3 | 今日转化 | 转化数 + 转化率 | 橙色 |
| 4 | 今日营收 | 营收金额 + 环比变化 | 紫色 |

**卡片样式**：

```
┌─────────────────────────────────┐
│  📖 今日阅读                    │
│                                 │
│      1,234                     │
│   ▲ 12.5% 较昨日               │
└─────────────────────────────────┘
```

**数据格式**：

| 指标 | 格式 | 示例 |
|------|------|------|
| 数值 | 千位分隔符 | 1,234 |
| 变化 | 箭头 + 百分比 | ▲ 12.5% |
| 负增长 | 红色箭头 | ▼ 5.2% |

### 3.2 趋势图表

**图表1：7日阅读量趋势**

- 类型：折线图
- X轴：日期（近7日）
- Y轴：阅读量
- 数据系列：各平台阅读量

**图表2：互动率趋势**

- 类型：折线图 + 目标线
- X轴：日期（近7日）
- Y轴：互动率%
- 目标线：3%

**图表3：营收趋势**

- 类型：柱状图
- X轴：日期（近7日）
- Y轴：营收金额

### 3.3 预警提示区

**位置**：首页右侧边栏

**展示内容**：

| 预警级别 | 图标 | 数量 | 触发条件 |
|----------|------|------|----------|
| 红色 | 🔴 | X篇 | 阅读<50 或 互动率<1% |
| 橙色 | 🟠 | X篇 | 阅读<100 或 互动率<2% |
| 黄色 | 🟡 | X篇 | 阅读<200 或 互动率<3% |

## 八、综合评分算法

### 8.1 评分维度

| 维度 | 权重 | 指标 |
|------|------|------|
| 流量 | 30% | 阅读量 |
| 互动 | 30% | 互动率 |
| 转化 | 25% | 转化率 |
| 收入 | 15% | 营收 |

### 8.2 评分计算

**步骤1：各维度标准化**

```
标准化分数 = (当前值 - 最小值) / (最大值 - 最小值) × 100
```

**步骤2：加权计算**

```
综合评分 = 流量分数 × 30% + 互动分数 × 30% + 转化分数 × 25% + 收入分数 × 15%
```

**步骤3：等级划分**

| 等级 | 分数范围 | 颜色 |
|------|----------|------|
| S | 90-100 | 金色 |
| A | 75-89 | 绿色 |
| B | 60-74 | 蓝色 |
| C | 40-59 | 黄色 |
| D | < 40 | 红色 |
