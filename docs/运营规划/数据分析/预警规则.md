# 预警规则文档

> 内容运营数据预警规则 - 四大维度全覆盖
> 版本：V2.0 | 创建日期：2026-02-27

---

## 一、预警体系概述

### 1.1 目的

建立覆盖四大维度（流量、互动、转化、收入）的数据预警规则，实现异常数据的及时发现和处理。

### 1.2 预警原则

| 原则 | 说明 |
|------|------|
| **及时性** | 7日内完成首次数据采集和预警判断 |
| **准确性** | 基于实际数据，避免主观判断 |
| **可操作** | 每个预警级别对应明确的响应措施 |
| **闭环管理** | 预警触发-分析-响应-跟踪形成闭环 |
| **四大维度** | 覆盖流量/互动/转化/收入 |

## 四、预警触发规则

### 4.1 触发条件

```
每日21:30 自动执行：
1. 扫描所有「tracking」状态的文章
2. 检查各维度指标是否触发阈值
3. 生成预警报告
4. 更新预警状态
```

### 4.2 预警ID命名规则

| 类型 | 格式 | 示例 |
|------|------|------|
| 阅读量预警 | WARN-V-{文章ID}-VIEW | WARN-V-TK-001-VIEW |
| 互动率预警 | WARN-V-{文章ID}-ENG | WARN-V-TK-001-ENG |
| 转化率预警 | WARN-V-{文章ID}-CONV | WARN-V-TK-001-CONV |
| 营收预警 | WARN-V-{文章ID}-REV | WARN-V-TK-001-REV |

### 4.3 预警记录字段

| 字段 | 说明 |
|------|------|
| 预警ID | WARN-V-{文章ID}-{类型} |
| 文章ID | 关联的文章ID |
| 预警类型 | 流量/互动/转化/收入 |
| 预警指标 | 阅读量/互动率/转化率/营收等 |
| 预警级别 | 红色/橙色/黄色 |
| 触发时间 | 首次触发预警的时间 |
| 触发数值 | 触发预警的具体数值 |
| 阈值 | 对应的阈值范围 |
| 处理状态 | 待处理/处理中/已解决/误报 |

## 五、预警响应措施

### 5.1 红色预警响应（紧急）

**触发条件**：
- 7日阅读 < 50
- 互动率 < 1%
- 转化率 < 1%
- 日营收 < 200

**响应时限**：24小时内启动分析

**响应措施**：

| 序号 | 措施 | 执行时间 | 负责人 |
|------|------|----------|--------|
| 1 | 紧急分析原因 | 24小时内 | CEO Agent |
| 2 | 检查内容质量 | 24小时内 | CEO Agent |
| 3 | 检查是否被限流 | 24小时内 | CEO Agent |
| 4 | 考虑重新编辑 | 48小时内 | CEO Agent |
| 5 | 优化标题/封面 | 48小时内 | CEO Agent |
| 6 | 追加推广资源 | 72小时内 | CEO Agent |
| 7 | 考虑撤回重发 | 7天后 | CEO Agent |

**分析清单**：
- [ ] 标题是否足够吸引人？
- [ ] 内容是否与平台调性匹配？
- [ ] 发布时间是否合适？
- [ ] 是否有违规被限流？
- [ ] 竞品同类内容表现如何？
- [ ] 是否存在技术问题（外链失效等）？

### 5.2 橙色预警响应（重要）

**触发条件**：
- 7日阅读 50~100
- 互动率 1%~2%
- 转化率 1%~2%
- 日营收 200~500

**响应时限**：48小时内启动分析

**响应措施**：

| 序号 | 措施 | 执行时间 | 负责人 |
|------|------|----------|--------|
| 1 | 分析原因 | 48小时内 | CEO Agent |
| 2 | 优化内容元素 | 72小时内 | CEO Agent |
| 3 | 监控后续表现 | 持续 | CEO Agent |

**分析清单**：
- [ ] 标题是否可以优化？
- [ ] 开头是否足够吸引？
- [ ] 内容结构是否清晰？
- [ ] CTA是否明确？
- [ ] 配图/排版是否合适？

### 5.3 黄色预警响应（关注）

**触发条件**：
- 7日阅读 100~200
- 互动率 2%~3%
- 转化率 2%~3%
- 日营收 500~1000

**响应时限**：每周复盘时处理

**响应措施**：

| 序号 | 措施 | 执行时间 | 负责人 |
|------|------|----------|--------|
| 1 | 记录并监控 | 每周 | CEO Agent |
| 2 | 总结经验 | 月度 | CEO Agent |
| 3 | 优化策略 | 月度 | CEO Agent |

## 六、预警关闭规则

### 6.1 关闭条件

| 条件 | 说明 |
|------|------|
| **自然恢复** | 30日内数据恢复正常水平 |
| **优化生效** | 采取措施后数据明显改善 |
| **内容撤回** | 内容被撤回或删除 |
| **误报排除** | 确认为数据延迟或统计问题 |
| **人工关闭** | 经分析确认无需继续跟踪 |

### 6.2 关闭流程

```
预警触发 → 原因分析 → 措施执行 → 效果跟踪 → 关闭预警
    ↑                                              ↓
    ←←←←←← 若效果不佳，重新分析 ←←←←←←←←←←←←←←←←
```
