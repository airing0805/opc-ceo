# 财务分类

## 概述

定义和维护收入与支出的分类体系，确保财务数据的一致性和可分析性。

## 收入分类

### INC-PROJ (项目收入)

来自项目开发的收入，包括：
- 外包开发
- 定制开发
- 技术服务

### INC-CONS (咨询服务)

咨询服务收入，包括：
- 技术咨询
- 方案设计
- 架构评审

### INC-PROD (产品销售)

产品销售收入，包括：
- 软件产品
- 课程/教程
- 数字资产

### INC-OTHER (其他收入)

其他来源收入，包括：
- 投资收益
- 奖金/补贴
- 其他非主要收入

## 支出分类

### EXP-OFF (办公设备)

办公设备支出，包括：
- 电脑硬件
- 显示器
- 外设（键盘、鼠标等）
- 办公家具

### EXP-SUB (软件订阅)

软件服务订阅，包括：
- 开发工具（IDE、数据库）
- SaaS 服务
- 云服务订阅
- 协作工具

### EXP-CLOUD (云服务)

云服务支出，包括：
- 服务器租用
- 数据库服务
- CDN
- 对象存储

### EXP-LEARN (学习培训)

学习与培训支出，包括：
- 在线课程
- 技术书籍
- 培训会议
- 认证考试

### EXP-TRAV (交通出行)

交通出行支出，包括：
- 差旅费用
- 通勤费用
- 其他交通

### EXP-OTHER (其他支出)

其他分类外支出，包括：
- 餐饮
- 娱乐
- 日常消费
- 其他杂项

## 分类扩展

### 添加新分类

当需要添加新分类时：

1. 确定分类类型（收入/支出）
2. 生成分类代码（3-4 字母缩写）
3. 定义分类说明
4. 更新本文档
5. 同步到所有相关模块

### 分类代码规范

| 规范 | 说明 |
|------|------|
| 格式 | `类型-缩写` |
| 类型 | `INC` (收入) 或 `EXP` (支出) |
| 缩写 | 3-4 个大写字母 |
| 示例 | `INC-PROJ`, `EXP-SUB` |

## 分类查询

### 按类型查询

```
查询所有收入分类
→ INC-PROJ, INC-CONS, INC-PROD, INC-OTHER

查询所有支出分类
→ EXP-OFF, EXP-SUB, EXP-CLOUD, EXP-LEARN, EXP-TRAV, EXP-OTHER
```

### 按关键词搜索

```
搜索"云"
→ EXP-CLOUD

搜索"培训"
→ EXP-LEARN
```

## MCP Memory 集成

### 分类实体存储

```yaml
entityType: FinancialCategory
name: INC-PROJ
observations:
  - type: income
  - code: INC-PROJ
  - name: 项目收入
  - description: 来自项目开发的收入
  - subcategories: [外包开发, 定制开发, 技术服务]
```

## 注意事项

- 分类代码必须唯一
- 分类变更需要更新所有关联的交易
- 删除分类前确认无关联交易
