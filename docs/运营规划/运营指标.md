# 运营指标

> 公司运营的关键绩效指标（KPI）定义
>
> **更新说明**：2026-02-25 扩展新增内容运营指标、变现指标、账号矩阵指标，以支持 Q2 知识付费+账号运营业务

---

## 指标分类

### 1. 任务完成指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 任务完成率 | 完成任务数 / 分配任务数 | ≥90% |
| 任务按时完成率 | 按时完成数 / 完成任务数 | ≥80% |
| 平均任务周期 | 任务完成时间总和 / 完成任务数 | <4小时 |

### 2. 沟通效率指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 用户响应时间 | CEO 响应用户的平均时间 | <30分钟 |
| 沟通文档覆盖率 | 有文档的沟通数 / 总沟通数 | 100% |
| 协调成功率 | 成功协调数 / 总协调数 | ≥90% |

### 3. 知识管理指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 知识记录数 | 每周新增知识实体数 | ≥5 |
| 知识利用率 | 被查询的知识数 / 总知识数 | ≥30% |
| 知识准确率 | 正确知识数 / 总知识数 | ≥95% |

### 4. 角色能力指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 角色参与度 | 各角色活跃天数 / 总天数 | 100% |
| 角色满意度 | 用户满意评分 | ≥4.5/5 |
| 角色成长度 | 新能力数 / 总能力数 | ≥10% |

### 5. 内容运营指标

> v4.1.1.1 - 追踪自媒体内容表现

| 指标 | 计算方式 | 目标值 | 备注 |
|------|----------|--------|------|
| 阅读量 | 内容实际阅读次数 | 日均≥1000 | 衡量内容曝光 |
| 点赞数 | 内容点赞总数 | 点赞率≥3% | 衡量用户认可 |
| 评论数 | 内容评论总数 | 评论率≥0.5% | 衡量用户参与 |
| 转发数 | 内容转发总数 | 转发率≥1% | 衡量内容传播 |
| 完播率 | 看完视频用户占比 | ≥50% | 视频内容专用 |
| 互动率 | (点赞+评论+转发)/阅读量 | ≥5% | 综合互动指标 |
| 粉丝增长数 | 净增粉丝数 | 月均≥500 | 账号成长追踪 |

### 6. 变现指标

> v4.1.1.2 - 追踪销售收入和转化效果

| 指标 | 计算方式 | 目标值 | 备注 |
|------|----------|--------|------|
| 销售收入 | 产品销售总金额 | 月均≥¥10000 | 核心变现指标 |
| 订单数 | 成交订单总数 | 月均≥50 | 订单量追踪 |
| 客单价 | 销售收入 / 订单数 | ≥¥200 | 平均订单金额 |
| 转化率 | 付费用户数 / 访客数 | ≥2% | 付费转化能力 |
| 课程销售占比 | 课程收入 / 总收入 | ≥60% | 产品结构分析 |
| 专栏销售占比 | 专栏收入 / 总收入 | ≥20% | 产品结构分析 |
| 会员销售占比 | 会员收入 / 总收入 | ≥10% | 产品结构分析 |
| 其他收入占比 | 其他收入 / 总收入 | ≤10% | 产品结构分析 |

### 7. 账号矩阵指标

> v4.1.1.3 - 追踪多平台账号运营健康度

| 指标 | 计算方式 | 目标值 | 平台 |
|------|----------|--------|------|
| 抖音粉丝数 | 抖音平台粉丝总数 | ≥10000 | 抖音 |
| 小红书粉丝数 | 小红书平台粉丝总数 | ≥5000 | 小红书 |
| 微信公众号粉丝数 | 微信公众号粉丝总数 | ≥3000 | 微信 |
| B站粉丝数 | B站平台粉丝总数 | ≥2000 | B站 |
| 抖音活跃度 | 月互动次数 / 粉丝数 | ≥5% | 抖音 |
| 小红书活跃度 | 月互动次数 / 粉丝数 | ≥8% | 小红书 |
| 微信公众号活跃度 | 文章打开率 | ≥10% | 微信 |
| B站活跃度 | 月互动次数 / 粉丝数 | ≥10% | B站 |
| 账号健康度 | (更新频率+互动率+粉丝增长)/3 | ≥70% | 综合评分 |

## 指标预警

| 指标 | 预警阈值 | 处理方式 |
|------|----------|----------|
| 任务完成率 | <80% | 分析阻塞原因，协调资源 |
| 用户响应时间 | >60分钟 | 检查 CEO 负载，考虑扩容 |
| 协调成功率 | <85% | 优化协调流程，增加沟通文档 |
| 互动率 | <3% | 分析内容质量，调整发布策略 |
| 转化率 | <1% | 优化产品定价和促销策略 |
| 账号健康度 | <60% | 检查更新频率，增加互动活动 |
| 粉丝增长数 | 负增长 | 分析内容方向，增加粉丝福利 |
