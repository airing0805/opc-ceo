# 内容生产规范

> Coach 制定规范，CEO 执行
> 更新日期：2026-02-25

---

## 核心原则

### 角色分工

| 角色 | 职责 | 执行方式 |
|------|------|----------|
| **Coach (我)** | 制定规范、规划选题、制定模板 | 对话中完成 |
| **CEO (你)** | 执行规范、通过任务分配生产内容 | 每日集中处理 |

### 执行规则

1. **对话中不执行** - 只规划、不动手
2. **每日集中处理** - 每天一次批量执行任务
3. **通过任务分配** - 使用 task-agent 或手动执行

---

## 内容生产流程

```
Coach 制定规范
       ↓
选题库规划（100篇）
       ↓
文章模板制定
       ↓
CEO 领取任务
       ↓
执行产出（task-agent 或 手动）
       ↓
每日集中处理发布
```

## 每日执行流程

1. **早晨**：CEO 领取当日任务（从选题库选1-2篇）
2. **执行**：使用 task-agent 或 手动写初稿
3. **傍晚**：集中发布到各平台
4. **复盘**：记录发布状态和数据
