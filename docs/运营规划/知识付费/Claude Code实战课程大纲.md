## 一、课程简介

### 1.1 课程定位

本课程是面向自媒体运营者和内容创作者的 Claude Code 实战入门课程，帮助学员掌握使用 Claude Code 进行自动化内容生产的核心技能。

### 1.2 目标受众

| 用户画像 | 描述 | 痛点 |
|----------|------|------|
| 自媒体新人 | 刚开始做自媒体，想提升效率 | 内容生产慢，不知道用什么工具 |
| 内容创作者 | 已有一定运营经验，想引入 AI | 听说过 AI 但不知道怎么用 |
| 一人公司创业者 | 独立运营，缺乏团队支持 | 时间精力有限，需要自动化 |
| 技术爱好者 | 对 AI 编程工具有兴趣 | 想学习新技术提升竞争力 |

### 1.3 学习目标

完成本课程后，学员将能够：

1. **独立配置环境** - 完成 Claude Code 的安装和基础配置
2. **开发简单 Agent** - 编写基础的自动化任务脚本
3. **使用 Skill 系统** - 创建和调用自定义技能模块
4. **集成外部工具** - 通过 MCP 协议连接各种外部服务
5. **构建实用工作流** - 实现周报生成、内容创作等自动化流程
6. **理解协作机制** - 掌握多 Agent 协作的基本原理
7. **部署自动化任务** - 将 Agent 部署到生产环境

### 1.4 前置要求

| 要求类型 | 具体内容 |
|----------|----------|
| 技术基础 | 了解基本的命令行操作 |
| 账号准备 | Claude 账号（Pro 或 Max） |
| 开发环境 | Node.js 18+、Git |
| 学习态度 | 愿意动手实践 |
