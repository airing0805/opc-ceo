# 表现评估

## 功能概述

定期审查 CEO 的决策质量、协调效率、上下文管理和角色运转状态，识别能力短板。

## 评估维度

### 1. 决策审查

**评估内容**：
- 决策是否有充分依据
- 决策结果是否符合预期
- 是否有遗漏的风险因素

**评估命令**：

```
审查决策:
- 时间范围: <YYYY-MM-DD 至 YYYY-MM-DD>
- 决策数量: <数量>
- 正确决策: <数量>
- 错误决策: <列表>
```

**示例**：
```
审查决策:
- 时间范围: 2026-02-16 至 2026-02-23
- 决策数量: 12
- 正确决策: 10
- 错误决策:
    - DEC-2026-02-20-003: 未考虑资源冲突
    - DEC-2026-02-22-001: 过度乐观估计时间
```

### 2. 效率评估

**评估内容**：
- 任务是否按时完成
- 是否有提前完成
- 延期原因分析

**评估命令**：

```
评估效率:
- 任务总数: <数量>
- 按时完成: <数量>
- 提前完成: <数量>
- 延期完成: <数量>
- 平均效率: <百分比>
```

### 3. 上下文检查

**评估内容**：
- 上下文是否完整
- 信息传递是否准确
- 是否有信息丢失

**评估命令**：

```
检查上下文:
- 会话数: <数量>
- 完整记录: <数量>
- 部分缺失: <数量>
- 信息失真: <数量>
```

### 4. 角色状态

**评估内容**：
- 各角色运转是否正常
- 是否有角色被闲置或过载
- 角色间协调是否顺畅

**评估命令**：

```
检查角色:
- 活跃角色: <列表>
- 问题角色: <列表>
- 协调问题: <列表>
```

## 评估周期

### 周评估（每周）

**时间**：每周日

**内容**：
1. 本周决策统计
2. 任务完成效率
3. 上下文质量
4. 角色运转状态
5. 生成周报

### 月评估（每月）

**时间**：每月最后一周

**内容**：
1. 月度决策趋势
2. 效率变化曲线
3. 能力成长轨迹
4. 阶段达标情况
5. 生成月报

### 阶段评估（每阶段末）

**时间**：基础期/成长期/成熟期结束时

**内容**：
1. 阶段目标完成度
2. 各维度达标情况
3. 成长速度评估
4. 是否进入下一阶段
5. 生成阶段报告

## 评估输出

### 评估报告格式

```markdown
# CEO 表现评估报告

**评估周期**: YYYY-MM-DD 至 YYYY-MM-DD
**当前阶段**: 基础期/成长期/成熟期

## 综合评分

| 维度 | 得分 | 等级 | 趋势 |
|------|------|------|------|
| 决策质量 | 85% | 合格 | ↑ |
| 协调效率 | 提前5% | 合格 | → |
| 上下文管理 | 完整 | 合格 | ↑ |
| 结果汇报 | 清晰 | 合格 | → |
| 角色管理 | 正常 | 合格 | → |

## 主要问题

1. <问题描述1>
2. <问题描述2>

## 改进建议

1. <建议1>
2. <建议2>

## 下周重点

1. <重点1>
2. <重点2>
```

## 无借口文化

**原则**：只看结果，不听解释

**执行方式**：
- 评估只关注数据和结果
- 不接受"因为..."开头的解释
- 问题就是问题，原因分析单独进行
- 改进必须体现在后续表现中

```

  "name": "eval-opc-ceo-core-2026-W08",
  "entityType": "Evaluation",
  "observations": [
    "period: 2026-02-17 至 2026-02-23",
    "type: weekly",
    "decisionQuality: 85%",
    "decisionQuality_level: 合格",
    "decisionQuality_trend: up",
    "coordinationEfficiency: 提前5%",
    "contextManagement: 完整",
    "reportQuality: 清晰",
    "roleManagement: 正常",
    "overallLevel: 合格",
    "issues: [决策时未考虑资源冲突]",
    "recommendations: [增加资源检查步骤]",
    "createdAt: 2026-02-23T18:00:00"
  ]
}
```

### 查询历史评估

```
使用 mcp__memory__search_nodes:
query: "Evaluation opc-ceo-core weekly"
```
