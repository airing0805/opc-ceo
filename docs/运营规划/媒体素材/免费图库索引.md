## 二、图库详细说明

### 2.1 Unsplash（首选推荐）

## 四、版权与授权说明

### 4.1 常见授权协议

| 协议 | 说明 | 商用 | 署名 |
|------|------|------|------|
| **CC0** | 完全放弃版权 | 可以 | 不需要 |
| **CC BY** | 署名即可使用 | 可以 | 需要 |
| **MIT** | 开源协议 | 可以 | 需要 |
| **Custom** | 自定义协议 | 需查看 | 需查看 |

### 4.2 使用规范

| 规范项 | 说明 |
|--------|------|
| **确认授权** | 使用前确认授权协议 |
| **记录来源** | 记录图片来源 URL |
| **署名要求** | CC BY 协议需署名 |
| **禁止事项** | 不使用禁止商用的素材 |

### 4.3 署名格式

```
图片来源: [图库名称]
作者: [作者名]
链接: [图片URL]
```

**示例**：
```
图片来源: Unsplash
作者: John Doe
链接: https://unsplash.com/photos/xxx
```
