# 素材复用规范

> 提高素材利用效率的复用规范
>
> 创建日期: 2026-02-27
>
> 版本: v1.0

---

## 一、复用原则

### 1.1 核心理念

```
一次制作，多处复用
降低成本，保持统一
```

### 1.2 复用优先级

| 优先级 | 策略 | 说明 |
|--------|------|------|
| P0 | **跨平台复用** | 同一内容多平台发布 |
| P1 | **同类主题复用** | 相似主题共享素材 |
| P2 | **模板化复用** | 基于模板快速生成 |
| P3 | **素材库复用** | 建立通用素材库 |

## 二、跨平台复用

### 2.1 一图多用矩阵

| 素材类型 | 知乎 | 简书 | CSDN | 掘金 |
|----------|------|------|------|------|
| 封面图 | 直接复用 | 直接复用 | 直接复用 | 直接复用 |
| 配图1 | 直接复用 | 直接复用 | 直接复用 | 直接复用 |
| 配图2 | 直接复用 | 直接复用 | 直接复用 | 直接复用 |
| 代码截图 | 可选 | 可选 | 必需 | 推荐 |

### 2.2 尺寸适配策略

| 原始尺寸 | 适配操作 | 适用平台 |
|----------|----------|----------|
| 1200x675 (16:9) | 无需调整 | 全平台通用 |
| 1080x1080 (1:1) | 裁剪或加白边 | 社交媒体 |
| 自定义 | 调整到 16:9 | 全平台 |

### 2.3 复用检查清单

- [ ] 尺寸是否兼容各平台
- [ ] 清晰度是否足够
- [ ] 文件大小是否合适（<500KB）
- [ ] 内容是否与平台风格匹配

## 四、模板化复用

### 4.1 模板复用流程

```
┌─────────────────────────────────────────────────────────┐
│ Step 1: 选择模板                                         │
│   - 根据文章类型选择封面模板                             │
│   - 参考封面模板库                                       │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ Step 2: 替换变量                                         │
│   - 主标题 → 文章标题                                    │
│   - 副标题 → 文章摘要/关键词                             │
│   - 图标 → 主题相关图标                                  │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────┐
│ Step 3: 导出复用                                         │
│   - 导出 16:9 通用尺寸                                   │
│   - 用于全平台发布                                       │
└─────────────────────────────────────────────────────────┘
```

### 4.2 模板变量说明

| 变量 | 说明 | 示例 |
|------|------|------|
| `{标题}` | 文章主标题 | 「Claude Code 入门」 |
| `{副标题}` | 文章简介 | 「AI 编程助手教程」 |
| `{标签}` | 分类标签 | 「AI」「教程」 |
| `{图标}` | 主题图标 | 终端图标 |
| `{配色}` | 主题配色 | 蓝紫色系 |

### 4.3 快速生成示例

**模板**：科技渐变风
**变量替换**：
- `{标题}` = 「Python 自动化入门」
- `{副标题}` = 「从零开始学自动化」
- `{图标}` = Python Logo
- `{配色}` = 蓝黄配色

**生成结果**：可用于全平台的封面图

## 五、素材库管理

### 5.1 素材存储结构

```
assets/
├── images/
│   ├── templates/           # 模板素材
│   │   ├── cover-tech.png   # 科技风模板
│   │   ├── cover-simple.png # 简约风模板
│   │   └── cover-code.png   # 代码风模板
│   ├── backgrounds/         # 背景素材
│   │   ├── gradient-1.png   # 渐变背景
│   │   └── gradient-2.png
│   ├── icons/               # 图标素材
│   │   ├── ai-icon.svg      # AI 图标
│   │   ├── code-icon.svg    # 代码图标
│   │   └── tool-icon.svg    # 工具图标
│   └── illustrations/       # 插画素材
│       ├── work-1.png       # 职场插画
│       └── study-1.png      # 学习插画
└── covers/                  # 已生成的封面
    ├── 2026-02/
    │   ├── cover-001.png
    │   └── cover-002.png
    └── ...
```

### 5.2 素材命名规范

| 类型 | 命名格式 | 示例 |
|------|----------|------|
| 模板 | `{类型}-{风格}.png` | `cover-tech.png` |
| 背景 | `{类型}-{序号}.png` | `gradient-1.png` |
| 图标 | `{主题}-icon.svg` | `ai-icon.svg` |
| 插画 | `{场景}-{序号}.png` | `work-1.png` |
| 已用封面 | `cover-{日期}-{序号}.png` | `cover-20260227-001.png` |

## 七、操作流程

### 7.1 发布前准备

| 步骤 | 操作 | 耗时 |
|------|------|------|
| 1 | 查找素材库是否有可复用素材 | 1分钟 |
| 2 | 如有 → 直接复用，微调标题 | 3分钟 |
| 3 | 如无 → 基于模板快速生成 | 5分钟 |
| 4 | 导出 16:9 通用尺寸 | 1分钟 |
| 5 | 用于全平台发布 | - |

### 7.2 素材库更新

| 周期 | 操作 | 说明 |
|------|------|------|
| 每周 | 整理新增素材 | 分类归档 |
| 每月 | 更新素材索引 | 记录使用次数 |
| 每季度 | 清理低效素材 | 删除不常用素材 |
