## 一、为什么需要思维导图？

### 1.1 大脑的工作方式

人的大脑天然是**网状思维**的：

```
传统线性笔记：
A → B → C → D → E

大脑真实思维：
    ┌─→ C ─→ E
    │
A ─→ B ─→ D
    │
    └─→ F ─→ G
```

思维导图正好符合大脑的**放射状思维**模式。

### 1.2 思维导图的价值

| 场景 | 无思维导图 | 有思维导图 |
|------|-----------|-----------|
| 读书笔记 | 看完就忘 | 形成知识体系 |
| 会议记录 | 信息零散 | 结构清晰 |
| 项目规划 | 漏东漏西 | 面面俱到 |
| 学习新知 | 碎片化 | 系统化 |
| 头脑风暴 | 想法混乱 | 井然有序 |

### 1.3 传统制作的痛点

传统思维导图制作的问题：

- 手动排版耗时（30分钟+）
- 调整结构麻烦
- 美化需要设计能力
- 修改一个牵动全局

**AI可以帮你解决这些问题：10秒生成，随时调整。**

## 三、Markdown思维导图格式

### 3.1 为什么用Markdown？

Markdown格式是**通用格式**，几乎所有思维导图工具都支持导入：

- Xmind：支持导入
- ProcessOn：支持导入
- MindManager：支持导入
- XMind：支持导入

### 3.2 Markdown思维导图语法

**基础语法**：

```markdown
# 中心主题

## 一级分支1
### 二级分支1-1
#### 三级分支1-1-1
#### 三级分支1-1-2
### 二级分支1-2

## 一级分支2
### 二级分支2-1
### 二级分支2-2
```

**缩进语法**（更常用）：

```markdown
# 中心主题

- 一级分支1
  - 二级分支1-1
    - 三级分支1-1-1
    - 三级分支1-1-2
  - 二级分支1-2
- 一级分支2
  - 二级分支2-1
  - 二级分支2-2
```

## 四、万能提示词模板

### 4.1 基础模板

```
请帮我生成一个思维导图，要求：
1. 主题：[主题名称]
2. 结构：[3-5个]一级分支
3. 每个一级分支下有[2-4个]二级分支
4. 输出格式：Markdown
5. 层级：[2-3层]
```

### 4.2 进阶模板

```
请帮我生成一个思维导图的Markdown格式内容，要求：

【主题】[核心主题]
【用途】[学习笔记/项目规划/知识梳理/其他]
【结构要求】
- 一级分支：[数量]个，围绕[方向]
- 二级分支：每个一级分支下[数量]个
- 三级分支：关键点细化
【内容要求】
- 简洁精炼，每条不超过15字
- 逻辑清晰，分类准确
- 覆盖全面，不遗漏重点
【输出格式】
使用Markdown缩进格式
```

### 4.3 高级模板（带示例）

```
请作为知识管理专家，帮我生成一个思维导图。

【背景】
我正在学习[主题]，需要建立一个系统的知识框架。

【目标】
- 理清[主题]的核心知识体系
- 找到学习路径
- 标记重点和难点

【结构要求】
1. 一级分支（4-6个）：
   - 涵盖[主题]的主要知识领域
   - 按照学习顺序或逻辑关系排列

2. 二级分支（每个一级下3-5个）：
   - 细化该领域的知识点
   - 标注【重点】或【难点】

3. 三级分支（可选）：
   - 具体概念或工具
   - 学习资源推荐

【输出格式】
Markdown缩进格式，示例：
# 主题
- 一级分支
  - 二级分支
    - 三级分支

请开始生成。
```


```markdown
# 《深度工作》读书笔记

## 核心概念
- 深度工作定义
  - 在无干扰状态下专注进行职业活动
  - 使个人的认知能力推向极限
- 浅浮工作定义
  - 在受到干扰的情况下开展的事务
  - 通常不会创造太多新价值
- 深度工作价值
  - 新经济中取得成功的必备能力
  - 能够掌握复杂工具
  - 能够在工作质量和速度方面达到精英层次

## 四种深度工作策略
- 禁欲主义哲学【难】
  - 完全屏蔽浅浮工作
  - 适合有高度自主权的人
- 双峰哲学
  - 时间分成深度工作和开放时间
  - 如：每周4天深度工作，1天开放
- 节奏哲学【推荐】
  - 每天固定时间深度工作
  - 如：每天上午8-12点
- 记者哲学【高难度】
  - 随时进入深度工作状态
  - 需要大量训练

## 实践方法
- 选择你的深度工作策略
- 建立深度工作仪式
  - 固定地点
  - 固定时间
  - 固定时长
  - 固定规则
- 像经营企业一样经营时间
  - 规划每一分钟
  - 设置截止日期
- 拥抱无聊
  - 训练专注力
  - 减少分心习惯

## 行动清单
- [ ] 确定每天深度工作时段
- [ ] 关闭社交媒体通知
- [ ] 建立固定工作仪式
- [ ] 每周复盘深度工作时间
- [ ] 设置有野心的目标
```

## 六、从Markdown到可视化

1. 复制AI生成的Markdown内容
2. 打开Xmind → 文件 → 导入 → Markdown

4. 调整样式和布局

### 6.2 导入到ProcessOn

**步骤**：

1. 登录ProcessOn
2. 新建 → 思维导图
3. 点击"更多" → 导入Markdown
4. 粘贴内容 → 生成导图

|------|----------|

## 七、高级技巧

### 7.1 动态调整

**场景**：思维导图需要修改或扩展

**方法**：直接告诉AI修改需求

```
请在上面的思维导图中：
1. 在"核心概念"分支下增加"应用场景"
2. 删除"历史背景"分支
3. 调整"实践方法"的顺序，把最重要的放在前面
```

### 7.2 合并多个导图

**场景**：有多个相关主题的思维导图需要合并

**方法**：

```
请合并以下两个思维导图，生成一个综合版本：

思维导图1：
[粘贴内容1]

思维导图2：
[粘贴内容2]

合并要求：
1. 去除重复内容
2. 合并相关分支
3. 保持逻辑清晰
```

### 7.3 导图转大纲

**场景**：需要把思维导图转成文章大纲

**方法**：

```
请将上面的思维导图转换为文章大纲，要求：
1. 一级分支作为章节标题
2. 二级分支作为小节标题
3. 三级分支作为内容要点
4. 补充每个小节的写作建议
```
