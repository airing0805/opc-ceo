## 搜索模式参考

| 模式 | 匹配 | 示例 |
|------|------|------|
| `*.md` | 所有 .md 文件 | `SKILL.md`, `设计意图.md` |
| `**/*.py` | 所有目录下的 .py | `src/main.py`, `utils/helper.py` |
| `file?.md` | file + 单字符 + .md | `file1.md`, `fileA.md` |
| `*[0-9]*.txt` | 包含数字的 .txt | `test1.txt`, `data2.txt` |

## 备份命令参考

### Windows (robocopy)

```
# 完整备份
robocopy "源目录" "目标目录" /E /XO /R:2 /W:0

# 增量备份
robocopy "源目录" "目标目录" /E /XO /MIR

# 仅验证（不实际复制）
robocopy "源目录" "目标目录" /L
```

### Linux/Mac (rsync)

```
# 完整备份
rsync -av --delete "源目录/" "目标目录/"

# 增量备份
rsync -av "源目录/" "目标目录/"

# 仅验证
rsync -avn "源目录/" "目标目录/"
```

## 常见问题

**Q: 如何搜索包含中文的文件？**
A: 直接使用中文字符串搜索，如 `按内容搜索: pattern:"任务管理"`

**Q: 如何避免文件覆盖？**
A: 创建文件时设置 `overwrite: false`，或先检查文件是否存在

**Q: 如何备份大目录？**
A: 使用增量备份（`incremental: true`），或分批次备份

**Q: 中文文件名在命令行中如何处理？**
A: 用引号包围路径，如 `"E:\Documents\设计意图.md"`
