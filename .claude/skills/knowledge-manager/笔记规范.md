# 笔记规范

## 功能概述

定义笔记的格式规范和模板。

## 笔记格式规范

### 基本格式

```markdown
# 标题

## 概述
<简要描述笔记的核心内容>

## 详细内容
<笔记的详细内容>

## 关键要点
- 要点1
- 要点2
- 要点3

## 相关资源
- [资源名称](链接)

## 标签
<标签列表>
```

# 2026-02-23 每日复盘

## 标签规范

### 按主题分类

| 标签类别 | 标签示例 |
|----------|----------|
| 技术 | 设计模式,算法,架构,优化 |
| 业务 | 需求,流程,规范 |
| 学习 | 读书,课程,实验 |
| 项目 | 项目A,项目B,里程碑 |
| 状态 | 待办,进行中,已完成 |

### 标签命名

- 使用简短中文
- 多个标签用逗号分隔
- 避免特殊字符

---

## 笔记组织原则

### 原则

1. **原子性** - 每个笔记聚焦一个主题
2. **可链接** - 笔记之间建立关联
3. **可检索** - 使用清晰的标题和标签
4. **可更新** - 保留修改历史
5. **可归档** - 定期归档已完成的内容

### 笔记生命周期

```
创建 → 持续更新 → 关联完善 → 归档
```
