# 工作日志体系

> 一人公司运营状态跟踪系统

---

## 体系架构

```
工作日志/
├── README.md              # 本文件 - 体系说明和模板
├── 日报/
│   └── README.md          # 每日工作记录
├── 周报/
│   └── README.md          # 每周总结
└── 月报/
    └── README.md          # 每月回顾
```

## 日志类型

| 类型 | 频率 | 用途 | 存储 |
|------|------|------|------|
| 日报 | 每日 | 记录当日工作详情 | `日报/` |
| 周报 | 每周 | 总结本周完成情况 | `周报/` |
| 月报 | 每月 | 回顾月度运营状态 | `月报/` |

## 填写规范

### 日报填写时机

- **会话结束时**：记录本次会话完成的工作
- **每日结束时**：汇总当日所有工作
- **任务完成时**：记录任务完成情况

### 周报填写时机

- **周日**：回顾本周完成情况，规划下周重点
- **与本周计划对照**：验证计划执行情况

### 月报填写时机

- **月末**：回顾本月运营状态
- **与战略目标对照**：验证目标进度

编辑 `月报/README.md`，添加新的月报内容。

## 自动化工具

### 周报自动生成

使用 Python 脚本自动从任务分配.md提取本周任务，生成周报：

```bash
# 生成当前周周报
python .claude/scripts/周报生成.py

# 指定周数
python .claude/scripts/周报生成.py --week 9 --year 2026

# 打印到控制台
python .claude/scripts/周报生成.py --print
```

**数据来源**:
- 任务分配.md - 任务ID、标题、状态、版本
- 日报/README.md - 每日工作回顾

**生成内容**:
- 本周概览（完成率、任务数）
- 任务完成统计（按状态、按类型）
- 本周完成任务清单
- 未完成任务清单
- 每日工作回顾
- 下周计划模板
