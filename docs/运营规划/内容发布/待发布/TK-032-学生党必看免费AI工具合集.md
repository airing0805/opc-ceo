## 二、学习必备免费AI工具

### 工具1：Kimi - 论文阅读神器

**一句话介绍**：上传PDF论文，AI帮你读懂核心内容。

**学生痛点**：
- 英文论文读不懂
- 长文献看不下去
- 重点找不到

**核心功能**：
- 上传PDF，一键总结
- 中英文翻译
- 提取关键信息
- 追问细节

**实战场景**：

```
场景：老师布置了一篇50页的英文文献

传统方式：
- 用翻译软件逐段翻译
- 看完就忘，找不到重点
- 耗时3小时以上

Kimi方式：
步骤1：打开Kimi，上传PDF
步骤2：输入："请用中文总结这篇论文的核心观点"
步骤3：30秒获得精炼摘要
步骤4：继续追问："研究方法是什么？"
步骤5：追问："结论对实际有什么启示？"

效果：15分钟掌握核心内容，还能应对课堂讨论
```

**使用技巧**：

```
论文阅读提示词模板：

1. 快速了解
   "请用300字总结这篇论文的核心观点"

2. 深入理解
   "请解释论文中的研究方法和实验设计"

3. 批判思考
   "这篇论文的局限性是什么？有什么可以改进的地方？"

4. 拓展学习
   "这个领域还有哪些重要的研究方向？"
```

**获取方式**：kimi.moonshot.cn

## 四、使用AI工具的正确姿势

### AI是辅助，不是替代

**错误认知**：让AI帮我写论文，我就不用写了
**正确认知**：AI帮我理清思路，提高效率，核心内容还是我来写

### 注意学术诚信

| 可以做 | 不可以做 |
|--------|---------|
| 用AI查找资料、理解概念 | 用AI直接生成论文提交 |
| 用AI润色语言、检查语法 | 不修改直接复制AI内容 |
| 用AI帮助理解作业要求 | 用AI直接做作业答案 |
| 用AI练习英语对话 | 用AI完成语言考试 |

### 保护个人信息

- 不要输入真实姓名、学号、身份证号
- 不要上传包含个人信息的文件
- 使用正规平台，避免山寨网站

## 五、常见问题解答

### Q1：免费版够用吗？

A：对学生来说，免费版完全够用。免费版的限制通常是"高频用户"才会遇到的问题。

### Q2：会不会被学校发现？

A：正常使用AI辅助学习不会被发现。关键是不要直接复制AI生成的内容，而是理解后用自己的话表达。

### Q3：用多了会不会变笨？

A：恰恰相反。用AI解决"低价值"任务（查资料、翻译），你可以把精力放在"高价值"任务（思考、创造）上。

### Q4：手机还是电脑？

A：推荐电脑端使用，屏幕大、操作方便。部分工具有手机APP，可以配合使用。
