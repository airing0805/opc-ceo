## 环境变量设置

### Windows PowerShell

```powershell
# 用户级别（永久）
[Environment]::SetEnvironmentVariable("变量名", "值", "User")

# 系统级别（永久，需管理员）
[Environment]::SetEnvironmentVariable("变量名", "值", "Machine")

# 临时（当前会话）
$env:变量名 = "值"

# 读取
[Environment]::GetEnvironmentVariable("变量名", "User")
$env:变量名

# 添加到 PATH
$currentPath = [Environment]::GetEnvironmentVariable("PATH", "User")
[Environment]::SetEnvironmentVariable("PATH", "$currentPath;新路径", "User")
```

### Linux/Mac Bash

```bash
# 临时
export VAR_NAME="value"

# 永久（用户级）
echo 'export VAR_NAME="value"' >> ~/.bashrc
source ~/.bashrc

# 添加到 PATH
export PATH="$PATH:/new/path"
echo 'export PATH="$PATH:/new/path"' >> ~/.bashrc
```

## 文件权限（Linux）

| 权限 | 数值 | 说明 |
|------|------|------|
| `---` | 0 | 无权限 |
| `--x` | 1 | 执行 |
| `-w-` | 2 | 写入 |
| `-wx` | 3 | 写入+执行 |
| `r--` | 4 | 读取 |
| `r-x` | 5 | 读取+执行 |
| `rw-` | 6 | 读取+写入 |
| `rwx` | 7 | 读取+写入+执行 |

| 常用 | 说明 |
|------|------|
| `755` | 目录：所有者全权限，其他读+执行 |
| `644` | 文件：所有者读写，其他只读 |
| `600` | 私密文件：仅所有者可读写 |
| `700` | 私密目录：仅所有者可访问 |
