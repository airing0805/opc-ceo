## 查询方法

### 1. 按日期查询

找到对应月份，按日期定位。

### 2. 按任务 ID 查询

在本页面搜索任务 ID（如 `v8.3.6`）

### 3. 按执行人查询

搜索执行人名称（`CEO` / `Coach`）

### 4. 按状态查询

- 搜索 `✅` - 已完成
- 搜索 `⚠️` - 部分完成
- 搜索 `❌` - 未完成

---

## 使用指南

### 创建新日志

1. 确认月份目录（如 `2026-03/`）
2. 复制 `执行日志模板.md`
3. 重命名为 `EXEC-YYYY-MM-DD-NNN.md`
4. 填写必填项（头部、目标、结果）
5. 更新本索引

### 更新统计

每月更新时，修改以下数值：
1. 月度统计中的执行次数
2. 年度统计表格中的对应行

**完成率计算**：
```
完成率 = (✅数 + ⚠️数 × 0.5) / 总数 × 100%
```
