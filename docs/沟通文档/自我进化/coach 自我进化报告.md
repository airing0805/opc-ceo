#### 3.1.1 主动风险预警系统

**改进内容**:
- 定义主动检测触发条件
- 建立周期性自动检测机制
- 增加风险量化评分

**实现方式**:
```markdown
主动风险检测时机：
├── 每次响应用户前（隐式检查）
├── 每次任务状态变更后
├── 每周定期全面检测
└── 用户明确要求时

风险评分公式：
风险分数 = 影响程度(1-5) × 发生概率(1-5) × 发现难度(1-5)
```
