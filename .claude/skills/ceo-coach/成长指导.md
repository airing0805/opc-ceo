# 成长指导

## 功能概述

基于评估结果触发成长流程，指导 CEO 能力更新，验证改进效果。

## 成长触发条件

### 自动触发

| 触发条件 | 阈值 | 触发动作 |
|----------|------|----------|
| 决策失误 | 连续2次 | 立即分析 + 指导 |
| 效率下降 | >20% | 原因分析 + 优化 |
| 评估不达标 | 任意维度 | 针对性指导 |
| 红线触碰 | 任意一条 | 替换评估 |

### 手动触发

| 触发条件 | 说明 |
|----------|------|
| 用户反馈 | 用户表达不满或建议 |
| 新需求 | 现有能力无法满足 |
| 主动请求 | CEO 请求帮助 |
| **深入分析** | 发现问题后需进行根因分析 |
| **模式提取** | 需要将问题转化为可复用的成长模式 |
