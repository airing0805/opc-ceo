## 2. 工具环境搭建

### 2.1 飞书多维表格创建

**步骤 1: 创建文档**

1. 打开飞书官网 (https://www.feishu.cn/)
2. 登录账号（推荐使用与项目管理相同的飞书账号）
3. 点击"新建" -> "多维表格"
4. 创建以下表格：

| 表格名称 | 用途 |
|----------|------|
| 每日内容数据 | 记录每日发布内容的数据 |
| 每日收入记录 | 记录每日订单和收入 |
| 每日粉丝数据 | 记录各平台粉丝变化 |
| 数据看板 | 汇总展示核心指标 |

**步骤 2: 配置字段**

参考《数据采集模板.md》中的字段配置，依次设置每个表格的字段。

**步骤 3: 设置视图**

为每个表格创建以下视图：

| 视图名称 | 用途 | 筛选条件 |
|----------|------|----------|
| 今日数据 | 查看今日录入 | 日期=今日 |
| 本周数据 | 本周汇总 | 日期>=本周一 |
| 历史数据 | 历史记录 | 日期<今日 |
| 异常数据 | 数据异常 | 环比>50% |

## 3. 数据更新提醒配置

### 3.1 飞书提醒设置

**方式一: 关注功能**

1. 打开已创建的多维表格
2. 点击右上角"分享" -> "关注"
3. 设置关注范围为需要提醒的字段

**方式二: 快捷方式**

1. 在移动端添加表格到"快捷方式"
2. 桌面端创建快捷方式到桌面

**方式三: 浏览器书签**

1. 在浏览器书签栏添加表格链接
2. 设置每日固定时间打开

## 5. 数据质量保障

### 5.1 数据校验规则

| 校验项 | 规则 | 处理方式 |
|--------|------|----------|
| 必填字段 | 所有字段不能为空 | 弹窗提醒 |
| 数值范围 | 数值>=0 | 弹窗提醒核实 |
| 趋势检查 | 环比变化>50% | 标记异常，需说明原因 |
| 目标检查 | 低于目标80% | 触发预警通知 |

### 5.2 数据异常处理

**常见异常类型**:

1. **数据突增/突降**
   - 原因: 可能是爆款内容或平台数据问题
   - 处理: 在备注中说明原因，记录异常

2. **数据缺失**
   - 原因: 忘记录入或平台数据延迟
   - 处理: 及时补录，标记为"补录"

3. **数据不一致**
   - 原因: 多平台数据统计口径不一致
   - 处理: 统一统计口径，以官方后台为准

## 6. 自动化配置（可选）

### 6.1 公式自动计算

在飞书多维表格中配置以下公式：

| 字段 | 公式 | 说明 |
|------|------|------|
| 净增粉丝 | 新增粉丝 - 取关粉丝 | 自动计算 |
| 互动率 | (点赞+评论+转发+收藏)/阅读量*100% | 自动计算 |
| 客单价 | 总收入/订单数 | 自动计算 |
| 目标完成率 | 当前值/目标值*100% | 自动计算 |

### 6.2 图表配置

创建数据看板时，添加以下图表：

| 图表类型 | 用途 | 数据源 |
|----------|------|--------|
| 折线图 | 收入趋势 | 每日收入记录 |
| 折线图 | 粉丝增长 | 每日粉丝数据 |
| 柱状图 | 内容互动 | 每日内容数据 |
| 饼图 | 收入构成 | 每日收入记录 |

## 8. 常见问题处理

### 8.1 飞书多维表格无法打开

**解决方案**:
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试使用客户端登录
4. 联系飞书客服

### 8.2 数据录入错误

**解决方案**:
1. 点击对应单元格直接修改
2. 或删除错误行重新录入
3. 在备注中说明修改原因

### 8.3 公式计算错误

**解决方案**:
1. 检查数据源是否为空
2. 检查数据类型是否正确
3. 确认公式语法正确
