## 五、内容制作标准

### 5.1 标题规范

| 规则 | 示例 |
|------|------|
| 数字+痛点/效果 | 5个AI工具让你的工作效率提升10倍 |
| 身份标签 | 程序员、宝妈、上班族 |
| 情绪词 | 必看、真实、揭秘、分享 |
| 避免标题党 | 适度夸张，避免夸大 |

### 5.2 文章结构

```
1. 开头（100-200字）
   - 痛点引入
   - 数据/案例开场

2. 主体（1500-2000字）
   - 方法1/步骤1
   - 方法2/步骤2
   - 方法3/步骤3

3. 结尾（200-300字）
   - 总结要点
   - 行动号召
   - 互动提问
```

### 5.3 配图要求

| 位置 | 要求 |
|------|------|
| 封面图 | 900x500像素，16:9比例 |
| 文中图 | 宽度600像素最佳 |
| 数量 | 每篇文章至少3张配图 |
| 风格 | 简洁、清爽、与内容相关 |
