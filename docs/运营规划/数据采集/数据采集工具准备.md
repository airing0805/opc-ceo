## 2. 需要采集的数据类型

| 数据类型 | 采集字段 | 采集频率 | 数据来源 |
|----------|----------|----------|----------|
| 阅读量 | 文章/视频阅读次数 | 每日 | 各平台后台 |

| 互动率 | 综合互动指标 | 每日 | 计算得出 |
| 粉丝增长 | 净增粉丝数 | 每日 | 各平台后台 |

| 数据类型 | 采集字段 | 采集频率 | 数据来源 |

| 订单数 | 成交订单总数 | 每日 | 支付平台/后台 |

| 转化率 | 付费用户/访客 | 每日 | 电商后台 |
| 产品销售结构 | 各产品线收入占比 | 每周 | 财务统计 |

### 2.3 账号矩阵数据

| 平台 | 采集字段 | 采集频率 | 数据来源 |
|------|----------|----------|----------|
| 微信公众号 | 粉丝数、阅读量、打开率 | 每日 | 公众号后台 |
| 抖音 | 粉丝数、播放量、互动率 | 每日 | 抖音后台 |
| 小红书 | 粉丝数、曝光量、收藏率 | 每日 | 小红书后台 |

## 3. 可用的采集工具与方法

### 3.1 平台官方方式

| 平台 | 采集方式 | 接入难度 | 优先级 |
|------|----------|----------|--------|
| 微信公众号 | 微信公众平台后台手动导出 | 低 | P0 |
| 抖音 | 抖音创作者服务平台数据导出 | 低 | P0 |
| 小红书 | 小红书专业号后台数据 | 低 | P0 |
| B站 | B站创作中心数据 | 低 | P0 |
| 知识付费平台 | 小鹅通/有赞后台 | 低 | P0 |

**说明**: 初期采用手动导出方式，后期可考虑API接入。

### 3.2 表格工具

| 工具 | 用途 | 适用场景 |
|------|------|----------|
| Excel | 日常数据记录 | 轻量级数据存储和计算 |
| Google Sheets | 多人协作数据 | 需要实时协作时 |
| 飞书多维表格 | 团队协作+自动化 | 日常数据更新 |

**推荐**: 使用飞书多维表格，支持：
- 自动提醒更新
- 数据可视化
- 多人协作
- 移动端查看

### 3.3 手动录入场景

对于无API的平台数据，采用手动录入：

| 数据类型 | 录入方式 | 录入频率 | 负责角色 |
|----------|----------|----------|----------|
| 公众号后台数据 | 每日截图+录入 | 每日 | opc-ceo-core |
| 抖音后台数据 | 每周导出Excel | 每周 | opc-ceo-core |
| 小红书后台数据 | 每周导出Excel | 每周 | opc-ceo-core |
| B站后台数据 | 每周导出Excel | 每周 | opc-ceo-core |
| 销售收入数据 | 每日从支付后台录入 | 每日 | opc-ceo-core |

## 5. 数据采集流程

### 5.1 每日数据采集流程

```
1. 早上9:00 - 打开数据采集表格
2. 登录各平台后台（公众号、抖音、小红书、B站）
3. 记录前一日核心数据：
   - 各平台粉丝数
   - 内容阅读量/播放量
   - 互动数据（点赞、评论、转发）
4. 登录支付后台，记录销售收入
5. 录入数据到飞书多维表格
6. 检查数据异常，设置预警
```

### 5.2 每周数据采集流程

```
1. 周日进行周度数据汇总
2. 导出各平台周报数据
3. 计算周度汇总指标：
   - 周粉丝增长
   - 周内容产出
   - 周收入汇总
4. 更新看板数据
5. 撰写周报分析
```
