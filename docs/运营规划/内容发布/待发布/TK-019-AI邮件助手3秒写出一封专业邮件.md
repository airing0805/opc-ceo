### 1.2 邮件写作的三大痛点

**痛点一：开头困难**

面对邮件不知道从何下笔：
- 过于正式还是随意？
- 如何称呼对方？
- 第一句话说什么？

**痛点二：措辞纠结**

同一个意思有无数种表达：
- "请查收"还是"请查阅"？
- "非常感谢"还是"感谢您"？
- 这个词会不会显得不专业？

**痛点三：格式混乱**

不同场景需要不同格式：
- 商务邮件怎么写？
- 内部邮件怎么写？
- 拒绝邮件怎么写？

**AI邮件助手可以帮你解决这三大痛点。**

## 三、五步快速回复法

### 步骤1：明确回复目的

在让AI写邮件前，先想清楚：

- 这封邮件要达到什么目的？
- 对方是谁？关系如何？
- 需要正式还是随意？

**目的分类**：
- 确认类：确认收到、确认时间、确认信息
- 请求类：请求帮助、请求批准、请求资源
- 回复类：回答问题、提供信息、解释说明
- 拒绝类：婉拒邀请、拒绝请求
- 跟进类：催促进度、提醒事项

### 步骤2：准备关键信息

AI需要的关键信息：

```
1. 对方身份：客户/同事/上级/下级
2. 邮件主题：关于什么
3. 回复要点：要说什么（3-5个要点）
4. 语气要求：正式/中性/亲切
5. 特殊要求：是否需要附件、是否需要结尾问候
```

### 步骤3：使用万能提示词

**基础版提示词**：

```
请帮我写一封邮件回复，要求：
1. 对方是[客户/同事/上级]
2. 回复要点：[要点1]、[要点2]、[要点3]
3. 语气：[正式/中性/亲切]
4. 字数：[100-200字]
```

**进阶版提示词**：

```
请帮我写一封专业的商务邮件回复，要求：

【收件人】XX公司张总（客户）
【邮件背景】对方询问项目进度
【回复要点】
1. 项目已完成80%
2. 预计下周三交付初版
3. 需要对方确认XX事项
【语气】专业、自信、友好
【特殊要求】
- 开头感谢对方关注
- 结尾表示如有问题随时沟通
- 字数200字左右
```

### 步骤4：快速审核修改

AI生成的邮件需要检查：

- [ ] 信息是否准确
- [ ] 语气是否恰当
- [ ] 是否有错别字
- [ ] 是否有敏感信息泄露

**审核技巧**：
- 关键数据务必人工核实
- 人名、公司名检查拼写
- 涉及金钱、时间的信息反复确认

### 步骤5：发送与追踪

发送后记得：
- 设置提醒追踪回复
- 重要邮件保存到草稿箱备用
- 建立个人邮件模板库

| 上级 | 正式、简洁 | "请您审批"、"汇报如下" |
| 客户 | 专业、友好 | "感谢您的信任"、"期待合作" |
| 同事 | 中性、高效 | "请查收"、"有劳" |
| 下级 | 亲切、明确 | "辛苦了"、"请注意" |
| 外部合作伙伴 | 商务、礼貌 | "冒昧打扰"、"感谢支持" |

### 5.2 邮件结构公式

**标准结构**：
```
称呼 + 开头（背景/目的） + 正文（要点） + 结尾（行动号召/感谢） + 落款
```

**长邮件结构**：
```
称呼
├── 开头：背景说明
├── 目录（可选）
├── 正文第一部分
├── 正文第二部分
├── 正文第三部分
├── 总结/下一步
└── 落款
```

### 5.3 避坑指南

**常见错误**：

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 过于随意 | 显得不专业 | 使用正式语气 |
| 信息过载 | 对方抓不住重点 | 分点陈述 |
| 模糊不清 | 来回沟通 | 明确具体 |
| 忘记附件 | 需要补发 | 发送前检查 |
