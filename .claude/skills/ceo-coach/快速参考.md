# 快速参考

## 能力标准速查

| 领域 | 合格 | 优秀 | 卓越 |
|------|------|------|------|
| 决策质量 | 80% | 90% | 95%+ |
| 协调效率 | 按时 | 提前10% | 提前20%+ |
| 上下文管理 | 完整 | 精准 | 预判 |
| 结果汇报 | 清晰 | 简洁 | 可直接汇报 |
| 角色管理 | 正常 | 优化 | 进化 |
| **教练能力** | 了解边界 | 精准分配、有效验收 | 主动识别缺口、触发进化 |
| **任务检查** | 基本验收 | 全面有标准 | 发现深层问题 |

## 成长时间窗口

| 阶段 | 时长 | 目标 | 不达标 |
|------|------|------|--------|
| 基础期 | 2周 | 掌握核心能力 | 警告 |
| 成长期 | 1个月 | 达到优秀 | 留用观察 |
| 成熟期 | 3个月 | 达到卓越 | 重新评估 |

## 淘汰红线（触碰即替换）

| # | 红线 | 说明 |
|---|------|------|
| 1 | 连续3次重大决策失误 | 决策能力严重不足 |
| 2 | 2周内无可见成长 | 学习能力不足 |
| 3 | 隐瞒问题或虚假汇报 | 诚信问题 |
| 4 | 推卸责任给下属角色 | 责任心不足 |
| 5 | 无法有效协调团队资源 | 协调能力不足 |
| 6 | 连续3次将任务分配给错误角色 | 教练能力不足 |
| 7 | 未能识别角色执行中的明显问题 | 检查能力不足 |
| 8 | 无法回答各角色的能力边界问题 | 边界理解不足 |

## 角色能力边界速查

| 角色 | 能做 | 不能做 |
|------|------|--------|
| task-manager | 任务CRUD、状态跟踪 | 文件操作、知识推理、财务 |
| file-manager | 文件搜索、操作、备份 | 任务状态、知识图谱 |
| knowledge-manager | 知识存储、检索、图谱 | 文件系统、财务 |
| finance-manager | 收入支出记录、报告 | 文件操作、任务管理 |
| schedule-manager | 事件创建、时间块 | 文件操作、财务计算 |
| automation-manager | 定时任务、工作流 | 直接执行业务逻辑 |
| claude-sdk-executor | API调用、批量处理 | 业务决策、用户交互 |

## 教练能力测试示例

```
Q: 用户要"找到上周修改的所有文档"，分配给哪个角色？
A: file-manager（文件搜索+时间过滤）

Q: task-manager 能直接修改文件吗？
A: 不能，需协调 file-manager

Q: 如何验收 file-manager 的"整理目录"任务？
A: 检查目录结构、命名规范、分类合理性
```
