## 关键决策

### 技术选型决策

| 领域 | 选型 | 理由 |
|------|------|------|
| Agent 框架 | Claude Agent SDK + 自研 | 与现有系统集成、无学习成本 |
| 数据存储 | SQLite + MCP Memory | 轻量、适合一人团队 |
| 任务调度 | APScheduler | Python 原生、简单易用 |
| 内容生成 | Claude API | 质量高、长文本支持好 |
| 自动化工具 | Playwright + requests | 跨浏览器、稳定 |

### 架构决策

| 决策 | 说明 |
|------|------|
| 单机部署优先 | MVP 阶段，降低复杂度 |
| Agent 协作模式 | CEO Agent 协调，专业 Agent 执行 |
| 平台适配器模式 | 统一接口，便于扩展新平台 |
| 数据分层存储 | 结构化数据 SQLite，知识数据 Memory |
