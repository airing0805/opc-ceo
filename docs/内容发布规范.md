## 二、CSDN 发布规范

### 标题格式

```
# 一文讲清 XXX（2026最火XXX）
# XXX 是什么？一篇文章讲清楚
```

### 内容结构

1. **前言**：一句话抓住注意力，说明本文解决什么问题
2. **比喻/类比**：用通俗的比喻讲清楚核心概念
3. **为什么重要**：讲清楚为什么读者需要关注
4. **怎么用**：实操步骤，代码+命令
5. **适合谁**：明确目标受众
6. **总结**：强化记忆
7. **下期预告**：引导关注

### 标签规范

```
MCP, Claude Code, AI编程, 2026, 入门教程
```

### 注意事项

- 代码块使用 ``` 标记
- 表格用 markdown 格式
- 重点句子加粗
- 每段不超过3行

## 四、内容生产流程

```
选题库 → 写初稿 → 平台适配 → 发布 → 数据跟踪
```

### 核心原则

1. **一次只写一篇**：不贪多
2. **模板化**：固定结构，只填变量
3. **少迭代**：初稿即终稿，微调就发
4. **实测为王**：只写自己用过的
