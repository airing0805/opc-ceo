## 二、Prompt 编写规范

### 2.1 通用 Prompt 结构

```
[主体描述] + [风格关键词] + [颜色/氛围] + [尺寸/构图]
```

### 2.2 风格关键词库

| 风格 | 关键词（中文） | 关键词（英文） |
|------|----------------|----------------|
| 扁平化 | 扁平化设计、极简风格 | flat design, minimalist |
| 科技感 | 科技感、未来感、数字风 | tech style, futuristic, digital |
| 插画 | 手绘插画、可爱风格 | illustration, hand-drawn, cute |
| 信息图 | 信息图、数据可视化 | infographic, data visualization |
| 商务 | 商务风格、专业感 | business style, professional |

### 2.3 颜色关键词

| 色系 | 中文关键词 | 英文关键词 |
|------|------------|------------|
| 蓝色系 | 蓝色系、科技蓝、渐变蓝 | blue tones, tech blue, blue gradient |
| 绿色系 | 绿色系、自然绿 | green tones, natural green |
| 暖色系 | 暖色系、橙黄色调 | warm colors, orange yellow tones |
| 中性色 | 灰白黑、简约配色 | neutral colors, black white gray |

### 2.4 配图 Prompt 模板

#### 技术教程配图

```
中文模板：
{主题} 技术概念图，扁平化设计，蓝色系，简约风格，无文字，16:9

英文模板：
{topic} technology concept illustration, flat design, blue tones, minimalist style, no text, 16:9 ratio
```

**示例**：
- 中文：`人工智能技术概念图，扁平化设计，蓝色系，简约风格，无文字，16:9`
- 英文：`AI technology concept illustration, flat design, blue tones, minimalist style, no text, 16:9 ratio`

#### 流程示意图

```
中文模板：
{流程描述} 流程图，信息图风格，图标+连线，简洁配色，1:1

英文模板：
{process} flowchart, infographic style, icons and arrows, clean colors, 1:1 ratio
```

#### 封面图

```
中文模板：
{主题} 封面图，{风格}，视觉冲击力，适合社交媒体，16:9

英文模板：
{topic} cover image, {style}, eye-catching, suitable for social media, 16:9 ratio
```

## 三、配图需求匹配

### 3.1 文章类型与配图风格

| 文章类型 | 推荐风格 | 工具推荐 | Prompt 要点 |
|----------|----------|----------|-------------|
| 技术教程 | 扁平化、流程图 | 通义万相 | 强调「教程」「步骤」 |
| 工具推荐 | 产品感、科技风 | Bing Creator | 英文 Prompt，强调「modern」 |
| 经验分享 | 插画、温馨风格 | 文心一格 | 中文 Prompt，情感关键词 |

## 四、配图质量检查清单

### 4.1 生成后检查

- [ ] 图片清晰度足够（无模糊、锯齿）
- [ ] 构图合理（主体居中或黄金分割）
- [ ] 色调统一（与文章风格一致）
- [ ] 无不适当内容
- [ ] 无明显 AI 瑕疵（手指、眼睛等）

### 4.2 使用前处理

| 处理项 | 工具 | 说明 |
|--------|------|------|
| 尺寸调整 | Tinypng、截图工具 | 调整到合适尺寸 |
| 压缩优化 | Tinypng | 减小文件大小 |
| 格式转换 | 在线工具 | 转为 JPG/PNG |

## 五、常见问题

### Q1: 生成图片有水印怎么办？

**解决方案**：
1. 优先选择无水印的免费工具（通义万相、文心一格）
2. 使用截图裁剪去除边缘水印
3. 更换其他工具生成

### Q2: 生成效果不满意？

**优化策略**：
1. 调整 Prompt 关键词
2. 尝试中英文切换
3. 生成多个候选，选择最佳
4. 切换其他 AI 工具

### Q3: 图片尺寸不合适？

**处理方法**：
1. 生成时指定尺寸比例
2. 使用截图工具裁剪
3. 使用在线工具调整
