## 7. 使用说明

### 7.1 每日记录流程

1. **采集时间**: 每日上午 9:00-9:30
2. **数据来源**: 各平台后台数据
3. **记录顺序**: 按平台依次记录昨日数据
4. **核对要求**: 确保数据准确无误

### 7.2 周汇总流程

1. **汇总时间**: 每周日晚 18:00-19:00
2. **统计周期**: 周一至周日
3. **计算内容**: 汇总各项指标，计算互动率
4. **备份要求**: 导出数据存档

### 7.3 月汇总流程

1. **汇总时间**: 每月1日上午
2. **统计周期**: 自然月
3. **对比分析**: 与上月数据对比
4. **报告输出**: 生成月度数据报告

## 8. 复制使用指南

### 8.1 复制到飞书多维表格

1. 选中上方表格内容
2. 复制（Ctrl+C）
3. 在飞书多维表格中粘贴（Ctrl+V）
4. 调整字段类型（日期、数字等）

### 8.2 复制到 Excel

1. 选中上方表格内容
2. 复制（Ctrl+C）
3. 在 Excel 中粘贴（Ctrl+V）
4. 设置单元格格式
