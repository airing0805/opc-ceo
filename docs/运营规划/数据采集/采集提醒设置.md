## 四、闹钟/提醒设置指引

### 手机闹钟设置（iOS）

```
1. 打开「时钟」App
2. 点击「闹钟」标签
3. 点击「+」添加新闹钟
4. 设置时间：21:30
5. 重复：每天
6. 标签：数据采集
7. 开启「铃声」和「振动」
```

### 手机闹钟设置（Android）

```
1. 打开「时钟」或「日历」App
2. 创建每日重复闹钟
3. 设置时间：21:30
4. 设置标签：数据采集
5. 开启提醒
```

### 电脑提醒设置（Windows）

```
1. 打开「任务计划程序」
2. 创建基本任务
3. 任务名称：每日数据采集提醒
4. 触发器：每日 21:30
5. 操作：显示消息（输入提醒内容）
```

### 浏览器插件（可选）

| 插件 | 用途 |
|------|------|
| Tabliss | 自定义每日提醒 |
| Notifier | 网页定时提醒 |

## 六、特殊情况处理

| 情况 | 处理方式 |
|------|---------|
| 忘记采集 | 次日 09:00 前补采，标注"补采" |
| 平台无法访问 | 记录 "N/A"，备注原因 |
| 临时有事 | 延后至 22:00 前完成 |
| 出差/外出 | 提前或延后采集，确保不中断 |
