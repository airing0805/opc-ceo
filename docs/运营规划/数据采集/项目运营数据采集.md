## 6. 数据采集命令集

### 6.1 Git 数据采集

```bash
# 统计本周提交数
git log --oneline --since="2026-02-23" --until="2026-02-26" | wc -l

# 统计本月提交数
git log --oneline --since="2026-02-01" --until="2026-02-28" | wc -l

# 查看提交类型分布
git log --format="%s" --since="2026-02-01" | cut -d: -f1 | sort | uniq -c | sort -rn

# 查看今日提交
git log --oneline --since="2026-02-25 00:00:00" --until="2026-02-25 23:59:59"
```
