## 实体定义规范

### 实体类型命名

- 使用 PascalCase
- 语义清晰
- 例如：`Task`, `Project`, `Transaction`, `Event`

### 实体属性规范

```json
{
  "name": "实体名称（唯一标识）",
  "entityType": "实体类型",
  "observations": [
    "属性1: 值1",
    "属性2: 值2",
    "createdAt: 2026-02-23T10:00:00",
    "updatedAt: 2026-02-23T10:00:00"
  ]
}
```

### 关系类型命名

- 使用 camelCase
- 动词+名词
- 例如：`belongsTo`, `dependsOn`, `createdBy`

## 版本控制

### 版本号格式

- 主版本.次版本.修订版本
- 例如：`1.0.0`

### 版本更新规则

- **主版本**：不兼容的重大变更
- **次版本**：向后兼容的功能新增
- **修订版本**：向后兼容的问题修复
