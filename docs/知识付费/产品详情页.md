- 税务管理（税费计算/税务提醒）
- 年度财务汇总

### 4. 内容生产管理模板（191行）

**核心功能**：选题管理、内容日历、多平台排期

**包含模块**：

**核心功能**：客户信息管理、跟进记录、交付进度追踪

**包含模块**：
- 客户信息总表（列表/类型/来源统计）
