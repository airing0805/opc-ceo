# 客户管理模板

## 模板用途说明

本模板专为**一人公司服务型 CEO** 设计，用于管理系统化客户关系。适用于咨询、培训、顾问、设计等服务类业务场景。

### 核心功能

- **客户信息管理** - 记录客户基本信息、联系方式、来源渠道
- **跟进记录** - 记录每次沟通的内容、承诺事项、下一步行动
- **交付进度追踪** - 跟踪项目/服务交付状态、关键里程碑、收款进度

## 七、模板使用说明

### 7.1 模板用途

本模板专为一人公司服务型CEO设计，用于系统化管理客户关系全生命周期。从客户信息录入、跟进记录到交付进度追踪、数据分析，帮助您在单兵作战的情况下也能提供专业级的客户服务。适用于咨询、培训、设计、顾问等服务类业务。

### 7.2 填写指导

**新增客户时**
1. 在"客户信息总表"添加基本信息
2. 复制"客户详细信息卡片"建立专属档案
3. 填写"需求画像"和"价值评估"
4. 确定客户类型和来源渠道

**每次沟通后**
1. 24小时内填写"跟进日志"
2. 记录沟通内容摘要和承诺事项
3. 明确下一步行动和跟进日期
4. 更新客户状态

**项目交付中**
1. 创建"交付里程碑详情"
2. 每完成一个里程碑更新状态
3. 同步更新"收款进度"
4. 里程碑延期需备注原因

**每周复盘**
1. 检查"客户健康度"
2. 跟进"休眠"和"风险"客户
3. 更新"月度跟进统计"

### 7.3 使用场景示例

| 场景 | 使用模块 | 操作要点 |
|------|----------|----------|
| 收到客户咨询 | 客户信息总表 | 快速录入基本信息 |
| 客户电话后 | 跟进记录表 | 记录要点，设提醒 |
| 项目启动 | 交付进度追踪 | 设里程碑，定收款计划 |
| 月度总结 | 数据统计与分析 | 统计转化率，分析漏斗 |

### 7.4 效果预期

坚持使用本模板2个月后，预期可实现：
- **客户信息零遗失** - 所有沟通有记录
- **跟进及时率提升至95%** - 明确的下次跟进日期
- **收款周期缩短30%** - 里程碑与收款绑定
- **转介绍率提升** - 系统化服务赢得口碑

**专业度提升技巧**：每次沟通前快速查看客户档案，让客户感受到被重视，建立信任基础。

### 7.5 更新频率建议

| 内容 | 更新时机 |
|------|----------|
| 客户信息 | 新增客户时填写 |
| 跟进记录 | 每次沟通后24小时内记录 |
| 交付进度 | 里程碑完成后更新 |
| 数据统计 | 每周/每月固定时间汇总 |

### 7.2 注意事项

1. **及时记录** - 每次沟通后尽快记录，避免遗漏关键信息
2. **定期回顾** - 每周检查一次"休眠"和"风险"客户，及时跟进
3. **隐私保护** - 客户信息仅用于业务管理，注意保密
4. **数据备份** - 定期备份客户数据，防止丢失

### 7.3 扩展建议

根据业务需要可扩展以下模块：

- 客户满意度调查
- 复购/续费提醒
- 转介绍奖励记录
- 竞争对手分析
- 合同/发票管理
