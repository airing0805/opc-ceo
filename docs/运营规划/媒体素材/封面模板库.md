## 二、封面模板设计

### 2.1 模板一：科技渐变风

**设计描述**：

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│    ████████████████████████████████████████████████     │
│    █                                              █     │
│    █    [图标/符号]                    {标题}     █     │
│    █                                              █     │
│    █                    {副标题/标签}              █     │
│    █                                              █     │
│    ████████████████████████████████████████████████     │
│                                                         │
│    背景装饰：几何图形、代码符号、网格线                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**配色方案**：

| 元素 | 颜色 | 色值 |
|------|------|------|
| 背景 | 深蓝到紫渐变 | #1a1a2e → #16213e |
| 标题 | 白色/亮色 | #ffffff |
| 图标 | 亮蓝色 | #4fc3f7 |
| 装饰 | 半透明几何 | rgba(255,255,255,0.1) |

**适用文章类型**：
- 技术教程
- 工具推荐
- AI 相关内容
- 编程学习

**生成方式**：
1. **AI 生成**：使用通义万相/Bing Creator，Prompt：`Technology cover image, dark blue gradient background, modern minimalist style, geometric decorations, 16:9 ratio`
2. **在线设计**：使用 Canva/创客贴，搜索「科技」「渐变」模板
3. **代码截图**：使用 Carbon 生成代码片段作为背景

**变体**：
- **蓝紫渐变**：适合 AI、编程主题
- **绿蓝渐变**：适合环保、新能源主题
- **橙红渐变**：适合热门、紧急主题

### 2.2 模板二：简约卡片风

**设计描述**：

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│    ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   │
│                                                         │
│         ┌─────────────────────────────────┐             │
│         │                                 │             │
│         │    [插画/图标]                   │             │
│         │                                 │             │
│         │         {标题}                  │             │
│         │                                 │             │
│         │       {分类标签}                │             │
│         │                                 │             │
│         └─────────────────────────────────┘             │
│                                                         │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**配色方案**：

| 元素 | 颜色 | 色值 |
|------|------|------|
| 背景 | 浅灰/米白 | #f5f5f5 / #fafafa |
| 卡片 | 白色 | #ffffff |
| 标题 | 深灰/黑色 | #333333 |
| 标签 | 品牌色 | #1976d2 |
| 插画 | 多彩 | 与主题相关 |

**适用文章类型**：
- 经验分享
- 职场话题
- 学习方法
- 生活感悟

**生成方式**：
1. **AI 生成**：使用文心一格，Prompt：`简约卡片风格封面，白色背景，中央卡片设计，扁平化插画，极简风格，16:9`
2. **在线设计**：使用 Canva/创客贴，搜索「简约」「卡片」模板
3. **手动组合**：白底 + 居中卡片 + Undraw 插画

**变体**：
- **暖色调**：适合生活、情感类
- **冷色调**：适合专业、职场类
- **多彩配色**：适合创意、设计类

### 2.3 模板三：代码主题风

**设计描述**：

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                         │
│    ┌──────────────────────────────────────────────┐    │
│    │  $ {命令行风格提示}                           │    │
│    │  > {代码关键词}                               │    │
│    │  ✓ {技术标签}                                 │    │
│    └──────────────────────────────────────────────┘    │
│                                                         │
│                    {大标题}                             │
│                                                         │
│                ━━━ {分隔线} ━━━                         │
│                                                         │
│                    {副标题}                             │
│                                                         │
│  // ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**配色方案**：

| 元素 | 颜色 | 色值 |
|------|------|------|
| 背景 | 深色终端风 | #1e1e1e / #282c34 |
| 代码块 | 半透明黑 | rgba(0,0,0,0.3) |
| 标题 | 绿色/白色 | #98c379 / #abb2bf |
| 命令提示符 | 绿色 | #98c379 |
| 注释 | 灰色 | #5c6370 |

**适用文章类型**：
- 编程教程
- 开发工具
- 代码实战
- 技术深度文

**生成方式**：
1. **Carbon 截图**：生成代码片段作为核心元素
2. **AI 生成**：使用通义万相，Prompt：`代码编辑器风格封面，深色背景，代码语法高亮，终端风格，科技感，16:9`
3. **在线设计**：使用 Canva，搜索「代码」「程序员」模板

**变体**：
- **VS Code 风格**：蓝绿色调
- **Terminal 风格**：黑绿配色
- **Sublime 风格**：橙黄色调

## 四、封面元素规范

### 4.1 文字规范

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 主标题 | 48-72px | 粗体 | 高对比度 |
| 副标题 | 24-32px | 常规 | 中对比度 |
| 标签/分类 | 16-20px | 常规 | 品牌色 |

### 4.2 图标规范

| 规范 | 说明 |
|------|------|
| 来源 | Iconfont / Flaticon / Undraw |
| 风格 | 与封面风格统一 |
| 颜色 | 与配色方案协调 |
| 位置 | 左上角或居中 |

### 4.3 排版原则

| 原则 | 说明 |
|------|------|
| **留白** | 至少 10% 边距 |
| **对比** | 文字与背景高对比 |
| **对齐** | 文字/元素对齐 |
| **层次** | 主标题 > 副标题 > 标签 |
