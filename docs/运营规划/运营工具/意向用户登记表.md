## 一、登记表说明

### 1.1 用途

用于收集首发活动早鸟价意向用户信息，便于：
- 上线第一时间通知意向用户
- 统计预热期用户积累情况
- 分析用户需求和行为

### 1.2 使用方式

1. 创建在线表单（如腾讯问卷、金数据等）
2. 将登记链接/二维码发布到各渠道
3. 定期导出数据进行统计分析
4. 上线时按登记顺序通知用户

## 七、操作流程

### 7.1 日常操作流程

1. **每日收集**（18:00）
   - 导出当日新增登记数据
   - 更新登记表
   - 统计当日新增数量

2. **每日统计**（20:00）
   - 更新每日统计表
   - 分析用户画像
   - 记录到执行日志

3. **上线通知**（03-07 08:00）
   - 按登记顺序通知用户
   - 标记通知状态
   - 跟踪购买状态

### 7.2 注意事项

- 微信号需核实格式正确
- 每日统计及时更新
- 通知时按登记顺序进行
- 前50名福利需要特别标记
