# 任务检查

## 功能概述

定义任务验收标准、检查流程和问题识别方法。

## 操作命令

### 命令 1：检查任务完成度

**用途**：检查任务是否全部完成

**语法**：
```
检查任务完成度:
- 任务ID: <任务标识>
- 执行角色: <角色名>
- 子任务: [子任务列表]
- 检查结果: <结果>
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| 任务ID | string | 是 | 任务唯一标识 |
| 执行角色 | string | 是 | 执行任务的角色 |
| 子任务 | array | 是 | 子任务完成情况 |
| 检查结果 | string | 是 | 完成度百分比 |

**示例**：
```
检查任务完成度:
- 任务ID: TASK-2026-02-23-001
- 执行角色: file-manager
- 子任务: [已整理文档, 已分类文件, 已清理重复]
- 检查结果: 100%
```

### 命令 2：检查任务准确性

**用途**：检查任务结果是否正确

**语法**：
```
检查任务准确性:
- 任务ID: <任务标识>
- 执行角色: <角色名>
- 预期结果: <预期>
- 实际结果: <实际>
- 检查结果: <结果>
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| 任务ID | string | 是 | 任务唯一标识 |
| 执行角色 | string | 是 | 执行任务的角色 |
| 预期结果 | string | 是 | 预期达到的结果 |
| 实际结果 | string | 是 | 实际得到的结果 |
| 检查结果 | string | 是 | 符合/不符合 |

**示例**：
```
检查任务准确性:
- 任务ID: TASK-2026-02-23-001
- 执行角色: file-manager
- 预期结果: 文档已按类型分类
- 实际结果: 文档已按类型分类到对应文件夹
- 检查结果: 符合
```

### 命令 3：全面验收

**用途**：从多个维度全面验收任务

**语法**：
```
全面验收:
- 任务ID: <任务标识>
- 执行角色: <角色名>
- 验收维度: [维度列表]
- 验收结果: <结果>
- 问题列表: [问题列表]
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| 任务ID | string | 是 | 任务唯一标识 |
| 执行角色 | string | 是 | 执行任务的角色 |
| 验收维度 | array | 是 | 完整性、准确性、规范性、效率、边界 |
| 验收结果 | string | 是 | 通过/不通过/需改进 |
| 问题列表 | array | 否 | 发现的问题列表 |

**示例**：
```
全面验收:
- 任务ID: TASK-2026-02-23-001
- 执行角色: file-manager
- 验收维度: [完整性, 准确性, 规范性, 效率, 边界]
- 验收结果: 通过
- 问题列表: []
```

## 检查维度

| 维度 | 检查内容 | 验收标准 |
|------|----------|----------|
| 完整性 | 任务是否全部完成 | 所有子任务100%完成 |
| 准确性 | 结果是否正确 | 符合预期结果 |
| 规范性 | 是否遵循规范 | 符合角色操作规范 |
| 效率 | 执行效率如何 | 在合理时间内完成 |
| 边界 | 是否越界操作 | 未超出角色边界 |

## 验收流程

```
1. 收集执行结果
   - 角色提交执行报告
   - CEO 收集所有相关输出

2. 对照验收标准
   - 逐条检查是否符合标准
   - 记录未达标项

3. 检查边界遵守
   - 确认未越界操作
   - 确认未遗漏职责

4. 评估执行质量
   - 优秀：超出预期
   - 合格：符合标准
   - 不合格：未达标

5. 反馈与改进
   - 合格：确认完成
   - 不合格：分析原因，安排重做或能力提升
```

## 问题识别与处理

| 问题类型 | 识别信号 | 处理方式 |
|----------|----------|----------|
| 能力不足 | 反复无法完成任务 | 触发能力提升流程 |
| 边界模糊 | 任务执行混乱 | 明确边界定义 |
| 标准不清 | 验收困难 | 细化验收标准 |
| 协作问题 | 多角色冲突 | 优化协调流程 |

## 验收结果等级

| 等级 | 说明 | 处理 |
|------|------|------|
| 优秀 | 超出预期，质量优秀 | 确认完成，记录优秀表现 |
| 合格 | 符合标准，无问题 | 确认完成 |
| 需改进 | 有小问题，可接受 | 指出问题，确认完成 |
| 不合格 | 未达标准，问题严重 | 安排重做或能力提升 |
