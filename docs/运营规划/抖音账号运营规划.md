## 二、内容类型规划

### 2.1 内容形式占比

| 维度 | 定位 |

| 风格 | 真实接地气、不装X |

| 参数 | 标准 |

| 比例 | 9:16 竖屏 |
| 帧率 | 30fps |
| 格式 | MP4 |
| 封面 | 文字标题+关键词 |

## 四、内容脚本框架

### 4.1 口播脚本模板

```
【开头3秒】hook
- 痛点/悬念/数据
- 例："90%的人做AI副业都赚不到钱，不是能力问题，而是..."

【中间】核心内容
- 观点/方法/案例
- 例："今天分享3个我用过的AI副业项目，第3个最简单..."

【结尾】引导互动
- 关注/评论/下期预告
- 例："下期讲如何从0到1开始做AI副业，想看的评论区扣1"
```

### 4.2 图文快闪模板

```
【画面1】关键词/痛点
【画面2-4】核心内容（3张）
【画面5】总结+引导
【配音】语速稍快、干货密集
```

### 4.3 混剪演示模板

```
【0-5秒】效果展示（Before/After）
【5-20秒】工具操作演示
【20-35秒】效果/收入证明
【35-40秒】引导关注
```
