## 1. 概述

本文档提供具体可用的数据采集模板，基于《数据看板模板》设计。模板适用于飞书多维表格或Excel，用于日常数据记录。

## 2. 内容运营数据采集模板

### 2.1 每日内容数据表

| 字段名 | 数据类型 | 说明 | 示例 |
|--------|----------|------|------|
| 日期 | 日期 | 采集日期 | 2026-02-25 |
| 平台 | 下拉选择 | 内容发布平台 | 微信公众号/抖音/小红书/B站 |
| 内容标题 | 文本 | 发布的内容标题 | MCP实战指南第一章 |
| 内容类型 | 下拉选择 | 内容形式 | 图文/短视频/长视频/文章 |
| 发布时间 | 时间 | 内容发布时间 | 14:30 |
| 阅读量/播放量 | 数值 | 内容曝光次数 | 1500 |
| 点赞数 | 数值 | 点赞数量 | 85 |
| 评论数 | 数值 | 评论数量 | 12 |
| 转发数 | 数值 | 转发/分享数量 | 8 |
| 收藏数 | 数值 | 收藏数量 | 25 |
| 完播率 | 百分比 | 视频看完比例(仅视频) | 65% |
| 互动率 | 百分比 | 综合互动指标 | 8.5% |
| 数据来源 | 文本 | 数据来源说明 | 抖音后台 |

**飞书多维表格配置示例**:
```
表格名称: 每日内容数据
字段设置:
- 日期: 日期类型，默认值=今日
- 平台: 单选"微信公众号","抖音","小红书","B站","知乎"
- 内容标题: 文本，单行
- 内容类型: 单选"图文","短视频","长视频","文章"
- 发布时间: 时间
- 阅读量/播放量: 数字，整数
- 点赞数: 数字，整数
- 评论数: 数字，整数
- 转发数: 数字，整数
- 收藏数: 数字，整数
- 完播率: 百分比
- 互动率: 公式=(点赞+评论+转发+收藏)/阅读量*100%
- 数据来源: 文本
```

## 5. 数据更新流程配置

### 5.1 每日更新流程

```
更新时间: 每日 9:00-9:30

Step 1: 打开数据采集表格
  └─ 飞书多维表格 -> 打开"运营数据采集"文档

Step 2: 录入内容数据
  └─ 打开"每日内容数据"表
  └─ 录入昨日发布内容的数据
  └─ 预计耗时: 5分钟

Step 3: 录入收入数据
  └─ 打开"每日收入记录"表
  └─ 录入昨日订单数据
  └─ 预计耗时: 3分钟

Step 4: 录入粉丝数据
  └─ 打开"每日粉丝数据"表
  └─ 登录各平台后台记录粉丝数
  └─ 预计耗时: 10分钟

Step 5: 检查数据完整性
  └─ 确认所有字段已填写
  └─ 检查数据异常（环比>50%）
  └─ 预计耗时: 2分钟
```

### 5.2 每周更新流程

```
更新时间: 每周日 18:00-19:00

Step 1: 汇总本周数据
  └─ 打开"内容汇总周表"
  └─ 自动计算本周累计数据

Step 2: 生成周报
  └─ 打开"周报汇总看板"
  └─ 填写本周分析摘要

Step 3: 数据备份
  └─ 导出本周数据到Excel存档
```

## 6. 表格关联配置

### 6.1 表间关联

| 源表 | 目标表 | 关联字段 | 用途 |
|------|--------|----------|------|
| 每日内容数据 | 内容汇总周表 | 日期/平台 | 周度汇总 |
| 每日收入记录 | 收入汇总表 | 日期 | 周期汇总 |
| 每日粉丝数据 | 账号矩阵汇总表 | 平台/账号 | 账号追踪 |

### 6.2 公式配置

**互动率计算**:
```
互动率 = (点赞数 + 评论数 + 转发数 + 收藏数) / 阅读量 × 100%
```

**客单价计算**:
```
客单价 = 销售总收入 / 订单数
```

**目标完成率**:
```
完成率 = 当前值 / 目标值 × 100%
```

## 7. 模板使用注意事项

1. **数据准确性**: 录入前需核对平台后台数据
2. **及时性**: 每日9:30前完成数据录入
3. **完整性**: 所有必填字段必须填写
4. **异常处理**: 数据异常时在备注中说明原因
