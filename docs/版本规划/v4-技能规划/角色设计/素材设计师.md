# 素材设计师角色设计

> 负责媒体素材的生成、查找和管理的专业角色
>
> 创建日期: 2026-02-26
>
> 状态: 规划中

---

## 一、角色定义

### 1.1 角色定位

| 维度 | 描述 |
|------|------|
| **角色名称** | 素材设计师 (Media Designer) |
| **所属系统** | V2 Agent 协作系统 |
| **核心职责** | 媒体素材的生成、查找、管理 |
| **服务对象** | 内容 Agent、运营 Agent |

### 1.2 角色价值

| 价值维度 | 说明 |
|----------|------|
| **效率价值** | 自动化素材准备，减少人工时间 |
| **质量价值** | 统一视觉风格，提升品牌形象 |
| **成本价值** | AI生成替代付费素材采购 |

## 三、工作流程

### 3.1 素材请求流程

```
1. 接收请求
   - 内容 Agent 发送配图需求
   - 包含：文章主题、配图位置、风格要求

2. 分析需求
   - 理解配图用途
   - 确定生成方式（AI生成/图库查找）
   - 设计 Prompt

3. 执行生成/查找
   - 调用 AI 模型生成
   - 或从免费图库查找

4. 返回结果
   - 提供 2-3 个候选
   - 包含素材文件和来源信息

5. 反馈优化
   - 根据反馈调整
   - 重新生成（如需要）
```

### 3.2 素材交付标准

| 要素 | 要求 |
|------|------|
| 格式 | PNG/JPG/WebP |
| 尺寸 | 宽度 ≥800px |
| 文件大小 | ≤2MB |
| 命名 | `{日期}-{序号}-{描述}.{格式}` |
| 元数据 | 包含来源、生成信息 |

### 4.1 核心技能

|------|------|------|
| AI 图像生成 | 高 | 熟练使用 Midjourney/DALL-E 等 |

| 视觉审美 | 中 | 具备基础审美判断 |
| 工具使用 | 中 | 熟悉图库、设计工具 |

### 4.2 AI 模型使用

| 模型 | 用途 | 调用方式 |
|------|------|----------|
| Midjourney | 高质量配图 | API/Discord |
| DALL-E 3 | 通用配图 | API |
| Stable Diffusion | 批量生成 | 本地/API |

## 七、绩效指标

### 7.1 效率指标

| 指标 | 目标 | 测量方式 |
|------|------|----------|
| 素材生成速度 | ≤10分钟/张 | 从请求到交付 |
| 一次通过率 | ≥80% | 无需返工的比例 |
| 日均产出 | ≥20张 | 每日生成素材数 |

### 7.2 质量指标

| 指标 | 目标 | 测量方式 |
|------|------|----------|
| 素材使用率 | ≥90% | 被实际使用的比例 |
| 风格一致性 | ≥95% | 符合风格指南的比例 |
| 版权合规率 | 100% | 无版权问题 |
