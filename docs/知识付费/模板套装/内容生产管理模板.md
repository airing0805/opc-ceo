# 内容生产管理模板

## 模板用途说明

本模板专为一人公司 CEO 设计，用于管理自媒体内容生产的全流程。适用于需要同时运营多个内容平台的个人创作者，帮助实现：

- **选题管理**：系统化收集、评估和筛选内容选题
- **日历规划**：可视化内容发布时间线
- **多平台排期**：协调不同平台的发布节奏
- **效率追踪**：监控内容生产进度和质量

### 1.3 选题状态流转

```
[灵感收集] → [初筛评估] → [列入计划] → [创作中] → [已完成] → [已发布]
    ↑                                                           |
    └────────────────── [未通过/搁置] ←────────────────────────┘
```

| 状态 | 说明 | 行动项 |
|------|------|--------|
| 灵感收集 | 初步想法，需要进一步验证 | 补充选题背景 |
| 初筛评估 | 评估可行性和价值 | 填写评估矩阵 |
| 列入计划 | 确认要做的选题 | 确认发布时间 |
| 创作中 | 正在进行内容创作 | 更新创作进度 |
| 已完成 | 内容已创作完成 | 等待发布排期 |
| 已发布 | 已发布到平台 | 记录发布数据 |
| 未通过 | 选题未通过审核 | 记录原因 |
| 搁置 | 暂时不做 | 重新评估时间 |

## 五、模板使用说明

### 5.1 模板用途

本模板是一人公司CEO的内容生产中枢，帮助您系统化管理自媒体内容从选题、创作到发布、追踪的全流程。适用于同时运营多平台（知乎、公众号、CSDN、小红书等）的个人创作者，实现内容生产标准化、发布节奏可控化。

### 5.2 填写指导

**周日规划（30分钟）**
1. 打开"月度内容主题"，确认本周主题
2. 在"选题库"中筛选3-5个选题
3. 使用"选题评估矩阵"打分排序
4. 填写"周度内容规划表"

**每日执行（创作时）**
1. 查看当日内容计划
2. 使用"内容创作Checklist"逐项完成
3. 创作完成后更新选题状态为"已完成"
4. 发布后填写"内容效果追踪表"

**每周复盘（20分钟）**
1. 统计本周发布数量和各平台数据
2. 分析"内容效果追踪表"找出优质内容特征
3. 调整下周选题方向

### 5.3 使用场景示例

| 场景 | 使用模块 | 操作要点 |
|------|----------|----------|
| 灵感爆发 | 选题来源追踪 | 快速记录，待筛选 |
| 不知写什么 | 选题评估矩阵 | 按得分高低排序 |
| 多平台发布混乱 | 多平台发布排期 | 按时间块分配平台 |
| 数据不理想 | 数据追踪表 | 分析原因，调整策略 |

### 5.4 效果预期

持续使用本模板1个月后，预期可实现：
- **选题不再迷茫** - 选题库提供持续灵感
- **发布节奏稳定** - 周计划保证稳定输出
- **多平台协调有序** - 清晰的发布排期
- **数据驱动优化** - 追踪表反馈创作方向

**效率倍增技巧**：核心内容一次创作，根据平台特性改编发布，实现"一鱼多吃"。

### 使用流程

1. **每周日**：规划下周内容主题，填写月度主题表
2. **每日早晨**：查看当日内容计划，更新选题状态
3. **创作过程中**：使用 Checklist 核对进度
4. **发布后**：记录数据，更新追踪表
5. **每周复盘**：分析数据，调整下周计划

### 一人公司特别提示

- **时间块管理**：建议按时间块分配不同类型内容创作
- **批量创作**：同类内容可批量创作，如周末集中写草稿
- **复用策略**：核心内容可改编后发布到不同平台
- **自动化辅助**：利用 AI 工具辅助创作和排版

### 模板更新建议

- 每周复盘时更新选题评估标准
- 每月末调整平台优先级
- 根据数据反馈优化内容类型比例
