## 二、执行追踪流程

### 2.1 选题→内容→发布流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   选题库    │───▶│  内容制作   │───▶│   发布上线  │───▶│  数据追踪  │
│  (491个)   │    │  进行中     │    │  已发布     │    │  复盘优化  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
      │                  │                  │                  │
      ▼                  ▼                  ▼                  ▼
   待处理           撰写中           公众号            阅读量
   进行中           审核中           视频号            点赞数
   已选用          已发布           B站               评论数
   已拒绝                          小红书            转化数
                                   抖音
```

### 2.2 状态定义

| 阶段 | 状态 | 说明 |
|------|------|------|
| **选题阶段** | 待处理 | 选题库中尚未被选中的选题 |
| | 已选用 | 已被选中待制作内容 |
| | 已拒绝 | 不适合当前制作 |
| **内容制作** | 撰写中 | 正在撰写内容 |
| | 审核中 | 内容审核/优化中 |
| | 已完成 | 内容制作完成待发布 |
| **发布阶段** | 已发布 | 内容已发布到平台 |
| | 待发布 | 排期待发布 |
| **数据追踪** | 监控中 | 发布后数据监控中 |
| | 已复盘 | 已完成数据复盘 |

## 四、执行标准

### 4.1 选题选用规则

| 维度 | 标准 | 权重 |
|------|------|------|
| 变现价值 | 高变现潜力选题优先 | 40% |
| 流量潜力 | 高热度选题优先 | 30% |
| 执行难度 | 制作简单优先 | 20% |
| 时效性 | 热点相关优先 | 10% |

### 4.2 内容制作标准

| 环节 | 标准 | 时长 |
|------|------|------|
| 选题确认 | 选定后24小时内确认 | - |
| 初稿撰写 | 图文1-2h，视频2-4h | 2h |
| 优化审核 | 30分钟-1h | 0.5h |
| 发布 | 排期后立即发布 | - |

### 4.3 发布平台优先级

| 优先级 | 平台 | 内容类型 | 说明 |
|--------|------|----------|------|
| P0 | 公众号 | 深度图文 | 私域核心 |

| P1 | B站 | 中长视频 | 专业形象 |
| P2 | 小红书 | 图文/短视频 | 种草引流 |
| P3 | 抖音 | 短视频 | 泛流量 |
