## 三、选工具的3个核心原则

### 原则1：先场景，后工具

**错误做法**：看到推荐就收藏，囤积一堆工具

**正确做法**：先明确自己的场景，再找对应工具

#### 场景-工具映射表

| 你的场景 | 对应类别 | 推荐工具 |
|---------|---------|---------|
| 每天要写公众号文章 | 内容创作 | Claude / ChatGPT |
| 需要开发一个小程序 | 编程开发 | Claude Code / Cursor |
| 要做产品宣传图 | 设计视觉 | Midjourney / Canva AI |
| 客户咨询太多回复不过来 | 效率工具 | Zapier + Claude |
| 需要做市场调研报告 | 数据分析 | Perplexity + Claude |
| 要写大量营销文案 | 营销运营 | Jasper / Copy.ai |

**一句话：不要为工具找场景，要为场景找工具。**

---

### 原则2：先免费，后付费

**错误做法**：直接订阅所有付费工具，结果用不上

**正确做法**：先用免费版验证价值，再决定是否付费

#### 工具免费版评估

| 工具 | 免费版能力 | 是否够用 |
|------|-----------|---------|
| ChatGPT | GPT-3.5，有限次数 | 基础问答够用 |
| Claude | 有限次数 | 轻度使用够用 |
| Kimi | 超长上下文，免费 | 完全够用 |
| Canva AI | 基础模板 | 够用 |
| Stable Diffusion | 完整功能 | 够用 |
| Zapier | 100次/月 | 轻度自动化够用 |

**付费决策清单**：

- [ ] 免费版是否已经满足80%需求？
- [ ] 付费功能是否解决核心痛点？
- [ ] 是否已经深度使用超过1个月？
- [ ] 计算ROI：节省的时间价值是否超过订阅费？

---

### 原则3：先精通，后扩展

**错误做法**：同时学习10个工具，每个都只懂皮毛

**正确做法**：先精通1-2个核心工具，再根据需要扩展

#### 学习路径建议

```
第1个月：精通1个核心工具
    ↓
第2-3个月：扩展到3个工具组合
    ↓
第4个月起：按需添加辅助工具
```

#### 精通标准

一个工具算"精通"需要：

- [ ] 知道它最适合解决什么问题
- [ ] 知道它的局限性在哪里
- [ ] 建立了自己的Prompt模板库
- [ ] 能在5分钟内完成常见任务
- [ ] 效率比手工提升3倍以上

### 信息来源

| 来源 | 类型 | 特点 | 推荐关注 |
|------|------|------|---------|

| **Hacker News** | 技术社区 | 前沿技术讨论 | 每周浏览 |

### 筛选策略

面对新工具，用这个流程快速判断：

```
1. 看定位 → 是否解决我的问题？
    ↓ 是
2. 看评价 → Product Hunt评分 > 4.0？
    ↓ 是
3. 看价格 → 免费版是否够用？
    ↓ 是
4. 试一试 → 用1周评估价值
    ↓ 有价值
5. 深入学习 → 纳入工具库
```

### 避免的坑

- ❌ 不要追新：新工具不一定比老工具好
- ❌ 不要贪多：工具数量与效率不成正比
- ❌ 不要迷信：任何工具都需要人工审核
- ❌ 不要停学：AI工具迭代很快，持续关注
