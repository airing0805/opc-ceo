# 任务模板

## 功能概述

提供任务模板功能，用于快速创建常见类型的任务，提高任务创建效率。

## 操作命令

### 命令 1：创建模板

**用途**：创建新的任务模板

**语法**：
```
创建模板:
  - name: <模板名称>
  - description: <模板描述>
  - fields:
      - title: <默认标题模板>
      - description: <默认描述模板>
      - priority: <默认优先级>
      - estimatedMinutes: <默认估算时间>
      - tags: <默认标签>
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| name | string | 是 | 模板名称（唯一） |
| description | string | 是 | 模板描述 |
| title | string | 否 | 默认标题模板，支持{date}变量 |
| description | string | 否 | 默认描述模板 |
| priority | string | 否 | 默认优先级：critical/high/medium/low |
| estimatedMinutes | number | 否 | 默认估算时间（分钟） |
| tags | string | 否 | 默认标签，逗号分隔 |

**示例**：
```
创建模板:
  - name: 每日检查
  - description: 每日例行检查任务模板
  - fields:
      - title: 每日检查 - {date}
      - priority: high
      - estimatedMinutes: 15
```

---

### 命令 2：使用模板创建任务

**用途**：使用模板快速创建任务

**语法**：
```
使用模板:
  - templateName: <模板名称>
  - overrides: (可选，覆盖模板字段)
      - title: <自定义标题>
      - projectId: <项目ID>
      - dueDate: <截止日期>
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| templateName | string | 是 | 模板名称 |
| overrides | object | 否 | 覆盖模板的字段 |

**示例**：
```
使用模板:
  - templateName: 每日检查
  - overrides:
      - projectId: PROJ-001
      - dueDate: 2026-02-24
```

---

### 命令 3：列出模板

**用途**：列出所有可用的任务模板

**语法**：
```
列出模板:
  - category: <分类> (可选)
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| category | string | 否 | 按分类过滤模板 |

**示例**：
```
列出模板:
  - category: 日常
```

---

```
  - templateName: <模板名称>
```

**参数**：

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| templateName | string | 是 | 模板名称 |

**示例**：

## MCP Memory 使用

### 创建模板实体

```
使用 mcp__memory__create_entities:
{
  "entities": [{
    "name": "TaskTemplate-每日检查",
    "entityType": "TaskTemplate",
    "observations": [
      "name: 每日检查",
      "category: 日常",
      "titleTemplate: 每日检查 - {date}",
      "description: 每日例行检查邮件、消息、待办",
      "defaultPriority: high",
      "defaultEstimatedMinutes: 15",
      "defaultTags: daily,routine",
      "createdAt: 2026-02-23T10:00:00"
    ]
  }]
}
```

### 查询模板

```
使用 mcp__memory__search_nodes:
{
  "query": "TaskTemplate category:日常"
}
```

### 删除模板

```
使用 mcp__memory__delete_entities:
{
  "entityNames": ["TaskTemplate-每日检查"]
}
```

| 开发 | 开发工作相关 | 代码审查、功能开发、bug修复 |
| 学习 | 学习和培训 | 课程学习、技术调研 |
| 其他 | 其他类型任务 | |

## 最佳实践

1. **模板复用** - 重复性任务使用模板提高效率
2. **合理分类** - 为模板设置清晰的分类便于查找
3. **灵活覆盖** - 使用模板时可以覆盖默认字段
4. **定期优化** - 根据实际使用情况优化模板内容
5. **标签规范** - 模板默认标签要规范，便于后续统计
