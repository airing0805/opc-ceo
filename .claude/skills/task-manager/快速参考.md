## 状态速查表

| 状态 | 含义 | 可转换到 |
|------|------|----------|
| `todo` | 待处理 | in_progress, cancelled |
| `in_progress` | 进行中 | completed, blocked, cancelled |
| `blocked` | 被阻塞 | todo, in_progress, cancelled |
| `completed` | 已完成 | （终态） |
| `cancelled` | 已取消 | （终态） |

## 任务示例

### 创建任务示例

```
创建任务:
  - title: 完成项目文档
  - description: 编写 OPC-CEO 项目的技术文档
  - priority: high
  - projectId: PROJ-001
  - tags: documentation,technical
  - dueDate: 2026-03-01
  - estimatedMinutes: 120
```

### 完成任务示例

```
完成任务:
  - taskId: TASK-2026-02-23-001
  - actualMinutes: 90
```

### 搜索任务示例

```
搜索任务:
  - filters:
      - status: todo
      - priority: high
  - sortBy: dueDate
  - sortOrder: asc
```

## 常见问题

**Q: 如何设置任务依赖？**
A: 使用"设置依赖"命令，指定 fromTaskId（依赖方）和 toTaskId（被依赖方）

**Q: 如何查看哪些任务被阻塞了？**
A: 搜索状态为 blocked 的任务，或使用"检查依赖"命令

**Q: 完成任务后会发生什么？**
A: 任务状态变为 completed，系统会自动检查并解除依赖此任务的其他任务

**Q: 如何使用模板快速创建任务？**
A: 使用"使用模板"命令，指定模板名称，可选择覆盖默认字段
