# 手动执行

## 执行流程

```
用户请求 → 确认任务 → 触发执行 → 监控状态 → 查看结果
```

## PowerShell 命令

```powershell
# 立即运行任务
Start-ScheduledTask -TaskName <TaskName>

# 查看任务状态
Get-ScheduledTaskInfo -TaskName <TaskName>
```

## 执行检查

- [ ] 任务已触发
- [ ] 状态变为 Running
- [ ] 日志开始记录
- [ ] 任务完成

## 查看执行结果

### 1. 查看日志

```powershell
Get-Content .claude\logs\<taskName>.log -Wait
```

### 2. 查看任务历史

```powershell
Get-ScheduledTaskInfo -TaskName <TaskName> | Select-Object -ExpandProperty LastRunTime
```

## 使用场景

| 场景 | 操作 |
|------|------|
| 测试任务配置 | 手动运行一次验证 |
| 立即执行某任务 | 不等待下次调度时间 |
| 调试任务问题 | 运行后查看日志 |
| 验证任务修复 | 重新运行确认问题解决 |
