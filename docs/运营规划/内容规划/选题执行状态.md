## 二、执行流程定义

### 2.1 完整执行流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   选题库    │───▶│   待写作    │───▶│   写作中    │───▶│   待审核    │───▶│   待发布    │
│  (491个)   │    │  已选用    │    │  撰写中    │    │  审核中    │    │  排期中    │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
                                                                    │
                                                                    ▼
                                                            ┌─────────────┐
                                                            │   已发布    │
                                                            │  上线完成   │
                                                            └─────────────┘
```

### 2.2 状态定义

| 状态 | 代码 | 说明 | 预计停留时间 |
|------|------|------|--------------|
| **待写作** | PENDING | 选题已选用，等待开始写作 | 1-2天 |
| **写作中** | WRITING | 正在撰写内容 | 2-4小时 |
| **待审核** | REVIEW | 内容初稿完成，等待审核优化 | 0.5-1小时 |
| **待发布** | READY | 内容审核通过，等待排期发布 | 按日历排期 |
| **已发布** | PUBLISHED | 内容已发布到平台 | - |

### 2.3 特殊状态

| 状态 | 代码 | 说明 |
|------|------|------|
| **暂停** | PAUSED | 临时暂停，后续继续 |
| **废弃** | DROPPED | 不再制作，标记废弃原因 |
