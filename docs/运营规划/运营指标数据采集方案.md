# 运营指标数据采集方案

> 自媒体运营数据采集的完整方案
>
> 任务 ID: TASK-2026-02-27-010
>
> 创建日期: 2026-02-25

---

## 1. 概述

### 1.1 方案目标

建立完整的数据采集体系，支撑自媒体运营决策，实现 2026 年收入 100 万的目标。

### 1.2 数据分类

| 数据类型 | 说明 | 优先级 |
|----------|------|--------|
| 内容数据 | 发布的文章、视频、选题相关数据 | P0 |
| 变现数据 | 收入、转化、付费相关数据 | P0 |
| 用户数据 | 粉丝增长、互动、行为相关数据 | P1 |
| 竞品数据 | 竞争对手、行业趋势相关数据 | P2 |

---

## 2. 内容数据采集

### 2.1 采集指标

| 指标名称 | 定义 | 计算方式 |
|----------|------|----------|
| 内容发布量 | 每周/每月发布的内容数量 | 统计发布内容数 |
| 内容类型分布 | 各类型内容占比 | 视频/图文/文章分类统计 |
| 发布频率 | 平均发布间隔 | 发布间隔时间均值 |
| 内容时长 | 视频/文章平均时长 | 总时长/内容数 |
| 选题命中率 | 爆款选题占比 | 10w+阅读选题数/总选题数 |
| 内容原创率 | 原创内容占比 | 原创内容数/总内容数 |

### 2.2 数据来源

| 数据源 | 采集方式 | 采集字段 |
|--------|----------|----------|
| 微信公众号 | 平台后台API | 阅读量、点赞、转发、在看 |
| 抖音 | 抖音数据API | 播放量、点赞、评论、完播率 |
| 小红书 | 小红书数据API | 曝光、收藏、点赞、评论 |
| B站 | B站数据API | 播放量、弹幕、点赞、投币 |
| 知乎 | 知乎数据API | 阅读量、赞同、收藏、评论 |
| 选题库 | 本地数据库 | 选题状态、分类、标签 |

### 2.3 采集频率

| 指标类型 | 采集频率 | 说明 |
|----------|----------|------|
| 发布数据 | 实时 | 内容发布后立即采集 |
| 互动数据 | 每日 | 每天固定时间批量采集 |
| 趋势数据 | 每周 | 周维度汇总分析 |
| 爆款数据 | 实时 | 达到阈值立即预警 |

### 2.4 分析方法

| 分析维度 | 方法 | 输出 |
|----------|------|------|
| 内容效果 | 归因分析 | 优质内容特征报告 |
| 选题效率 | 漏斗分析 | 选题转化率排名 |
| 发布节奏 | 时序分析 | 最佳发布时间建议 |
| 类型对比 | A/B测试 | 内容类型效果对比 |

---

## 3. 变现数据采集

### 3.1 采集指标

| 指标名称 | 定义 | 计算方式 |
|----------|------|----------|
| 总收入 | 所有渠道收入总和 | 汇总各渠道收入 |
| 收入增长率 | 环比增长比例 | (本期-上期)/上期×100% |
| 客单价 | 平均每单收入 | 总收入/订单数 |
| 转化率 | 付费用户/总用户 | 付费用户数/总用户数 |
| 付费用户数 | 累计付费用户 | 统计各渠道付费用户 |
| ARPU | 每用户平均收入 | 总收入/总用户数 |
| 复购率 | 重复付费比例 | 复购用户数/付费用户数 |
| 渠道收入占比 | 各渠道收入占比 | 渠道收入/总收入 |

### 3.2 数据来源

| 数据源 | 采集方式 | 采集字段 |
|--------|----------|----------|
| 知识付费平台 | 平台API/后台 | 课程销售、专栏订阅 |
| 广告联盟 | 联盟后台API | 广告展示、点击、收入 |
| 社群平台 | 社群管理后台 | 群成员、付费、续费 |
| 电商平台 | 电商后台API | 商品销售、佣金收入 |
| 支付系统 | 支付网关API | 订单金额、支付状态 |
| Excel/表格 | 手动录入 | 补充未接入API的数据 |

### 3.3 采集频率

| 指标类型 | 采集频率 | 说明 |
|----------|----------|------|
| 交易数据 | 实时 | 支付成功后立即采集 |
| 收入汇总 | 每日 | 每日0点汇总昨日收入 |
| 财务对账 | 每周 | 周维度财务核对 |
| 月度结算 | 每月 | 月度收入报表 |
| 年度复盘 | 每年 | 年度收入分析 |

### 3.4 分析方法

| 分析维度 | 方法 | 输出 |
|----------|------|------|
| 收入构成 | 构成分析 | 收入来源占比报告 |
| 增长趋势 | 趋势分析 | 月度/季度增长曲线 |
| 渠道效果 | 归因分析 | 渠道ROI排名 |
| 用户价值 | RFM分析 | 用户分层运营建议 |
| 定价策略 | 价格测试 | 最优定价方案 |

## 4. 用户数据采集

### 4.1 采集指标

| 指标名称 | 定义 | 计算方式 |
|----------|------|----------|
| 总粉丝数 | 各平台粉丝总和 | 跨平台粉丝汇总 |
| 新增粉丝 | 期间新增粉丝数 | 期末-期初粉丝数 |
| 粉丝增长率 | 粉丝增长速度 | 新增粉丝/期初粉丝×100% |
| 活跃粉丝 | 近期互动粉丝 | 30天内有互动行为 |
| 互动率 | 互动用户/总用户 | 点赞+评论+转发/粉丝数 |
| 留存率 | 粉丝留存比例 | 期末仍关注/期初关注 |
| 用户画像 | 粉丝特征分布 | 地域/年龄/性别/兴趣 |
| 用户来源 | 粉丝来源渠道 | 各渠道引流占比 |

### 4.2 数据来源

| 数据源 | 采集方式 | 采集字段 |
|--------|----------|----------|
| 微信公众号 | 平台后台API | 用户画像、增减趋势 |
| 抖音 | 抖音数据API | 粉丝画像、作品数据 |
| 小红书 | 小红书数据API | 粉丝数据、互动数据 |
| B站 | B站数据API | 粉丝数据、投币数据 |
| 知乎 | 知乎数据API | 关注者数据 |
| 问卷调查 | 在线表单 | 用户兴趣、需求调研 |
| 社群 | 社群管理工具 | 群成员互动数据 |

### 4.3 采集频率

| 指标类型 | 采集频率 | 说明 |
|----------|----------|------|
| 粉丝数据 | 每日 | 每日同步各平台粉丝数 |
| 互动数据 | 每日 | 每日采集互动行为 |
| 用户画像 | 每周 | 周维度更新画像 |
| 问卷调查 | 每月 | 月度用户调研 |
| 留存分析 | 每月 | 月度留存报表 |

### 4.4 分析方法

| 分析维度 | 方法 | 输出 |
|----------|------|------|
| 增长分析 | 漏斗分析 | 增长转化路径优化 |
| 活跃分析 | 群组分析 | 活跃用户特征报告 |
| 留存分析 | 队列分析 | 留存率变化趋势 |
| 用户分层 | RFM模型 | 用户价值分层 |
| 内容匹配 | 推荐分析 | 用户兴趣内容匹配 |
