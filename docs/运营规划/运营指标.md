# 运营指标

> 公司运营的关键绩效指标（KPI）定义

---

## 指标分类

### 1. 任务完成指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 任务完成率 | 完成任务数 / 分配任务数 | ≥90% |
| 任务按时完成率 | 按时完成数 / 完成任务数 | ≥80% |
| 平均任务周期 | 任务完成时间总和 / 完成任务数 | <4小时 |

### 2. 沟通效率指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 用户响应时间 | CEO 响应用户的平均时间 | <30分钟 |
| 沟通文档覆盖率 | 有文档的沟通数 / 总沟通数 | 100% |
| 协调成功率 | 成功协调数 / 总协调数 | ≥90% |

### 3. 知识管理指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 知识记录数 | 每周新增知识实体数 | ≥5 |
| 知识利用率 | 被查询的知识数 / 总知识数 | ≥30% |
| 知识准确率 | 正确知识数 / 总知识数 | ≥95% |

### 4. 角色能力指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 角色参与度 | 各角色活跃天数 / 总天数 | 100% |
| 角色满意度 | 用户满意评分 | ≥4.5/5 |
| 角色成长度 | 新能力数 / 总能力数 | ≥10% |

---

## 指标监控

### 每日监控

- [ ] 任务完成数
- [ ] 用户响应时间
- [ ] 沟通文档更新数

### 每周复盘

- [ ] 任务完成率
- [ ] 沟通效率
- [ ] 知识管理效果

### 每月分析

- [ ] 整体运营健康度
- [ ] 角色能力成长
- [ ] 运营优化方向

---

## 指标预警

| 指标 | 预警阈值 | 处理方式 |
|------|----------|----------|
| 任务完成率 | <80% | 分析阻塞原因，协调资源 |
| 用户响应时间 | >60分钟 | 检查 CEO 负载，考虑扩容 |
| 协调成功率 | <85% | 优化协调流程，增加沟通文档 |
