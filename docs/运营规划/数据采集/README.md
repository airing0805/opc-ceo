## 采集操作流程

### 每日采集流程 (21:00-22:00)

1. **内容数据采集** (21:00)
   - 登录各平台后台
   - 记录当日发布内容的数据
   - 更新 `内容数据采集/README.md`

2. **用户数据采集** (21:30)
   - 记录各平台粉丝数变化
   - 更新 `用户数据采集/README.md`

3. **变现数据采集** (22:00)
   - 检查各变现渠道收入
   - 更新 `变现数据采集/README.md`

### 每周采集流程 (周五 18:00)

1. 汇总本周数据
2. 计算周环比增长
3. 更新项目运营数据采集文档

### 每月采集流程 (月末最后一天)

1. 汇总月度数据
2. 对照月度目标完成率
3. 生成月度分析报告
