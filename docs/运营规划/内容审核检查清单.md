## 一、内容完整性检查（5项）

- [ ] **主题明确**：文章有清晰的核心主题，标题准确反映内容
- [ ] **结构完整**：包含引言、主体、结论三部分，逻辑清晰
- [ ] **内容充实**：主体内容有足够的深度和细节，不空洞
- [ ] **信息准确**：引用数据、案例、来源可靠准确
- [ ] **结尾有力**：有总结或行动号召，不虎头蛇尾

---

## 二、格式规范检查（5项）

- [ ] **标题规范**：标题层级清晰（H1/H2/H3），无语法错误
- [ ] **段落分明**：段落长度适中，避免长段堆砌
- [ ] **标点正确**：使用规范标点，无错用、滥用标点
- [ ] **空格规范**：中英文之间、数字与单位之间有空格
- [ ] **格式统一**：全文格式风格一致

---

## 三、敏感信息检查（4项）

- [ ] **无个人隐私**：不包含个人身份证号、手机号、地址等隐私信息
- [ ] **无敏感词汇**：不含政治敏感、色情暴力、违法犯罪等内容
- [ ] **无商业机密**：不泄露公司内部信息、客户商业秘密
- [ ] **无侵权内容**：不侵犯他人著作权、商标权、肖像权

---

## 四、平台合规检查（4项）

- [ ] **平台规则**：符合目标平台的发布规范和社区准则
- [ ] **字数合适**：符合平台字数要求，不过长或过短
- [ ] **标签准确**：添加了合适的分类标签，便于推荐
- [ ] **无导流信息**：不含外链、微信二维码等导流内容（除非平台允许）

---

## 五、内容质量检查（4项）

- [ ] **价值明确**：读者能从中获得有价值的信息或启发
- [ ] **语言流畅**：表达清晰准确，无语法错误和错别字
- [ ] **可读性强**：结构合理，阅读体验良好
- [ ] **原创性**：内容为原创或已注明转载来源
