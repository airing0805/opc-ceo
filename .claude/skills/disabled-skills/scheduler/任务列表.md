# 任务列表

## 查询流程

```
用户请求 → 读取 Memory → 过滤状态 → 生成列表
```

## 查询方式

### 1. 列出所有任务

```powershell
Get-ScheduledTask | Where-Object {$_.TaskName -like "*"}
```

### 2. 列出活动任务

```powershell
Get-ScheduledTask | Where-Object {$_.State -eq "Ready"}
```

### 3. 查看任务详情

```powershell
Get-ScheduledTaskInfo -TaskName <TaskName>
```

## 输出格式

| 任务名称 | 类型 | 调度 | 状态 | 下次运行 |
|---------|------|------|------|---------|
| CoachSelfEvolution | 循环 | 每20分钟 | Ready | 12:20 |
| DailyBackup | 循环 | 每天02:00 | Ready | 02:00 |
| OneTimeTask | 一次性 | 已执行 | Disabled | - |

## 任务状态说明

| 状态 | 说明 |
|------|------|
| Ready | 任务已就绪，等待执行 |
| Running | 任务正在执行 |
| Disabled | 任务已禁用 |
| Deleted | 任务已删除 |

## Memory 查询

从 MCP Memory 查询 Task 实体：

```
搜索类型: Task
过滤条件:
  - status = active
  - status = disabled
  - status = deleted
```
